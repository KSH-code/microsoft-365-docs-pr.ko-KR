## <a name="europe-the-middle-east-and-africa-emea"></a>EMEA(유럽, 중동, 아프리카)

EMEA 지역의 Microsoft 365 서비스를 구입하는 경우 이 구매에는 VAT(Value-Added 부가가치세)가 적용될 수 있습니다.
  
- 유럽 연합 회원국, 아르메니아, 벨로루시, 노르웨이, 터키 또는 아랍에미리트에 있는 경우 유효한 로컬 VAT ID를 제공하지 않는 경우 Microsoft Ireland Operations Ltd.는 계정이 설정된 청구 국가에 따라 현재 현지 VAT 세율을 적용합니다.

- 리히텐스테인, 러시아, 세르비아, 남아프리카 공화국 또는 스위스에 있는 경우 VAT ID 제공 여부에 따라 VAT가 적용됩니다.

- EU 고객의 경우 VAT ID의 유효성을 검사하려면 먼저 VIES(VAT Information Exchange System)에서 해당 ID를 검증할 수 있어야 합니다. VAT ID를 확인할 수 없는 경우 현지 세무서에 문의합니다.

다음 경우 VAT 0 등급을 받을 수 있습니다.
  
- **아일랜드 외부의 유럽 연합 회원국에 있습니다.** 유효한 로컬 VAT ID를 제공할 수 있습니다. 이를 통해 Microsoft Ireland Operations Ltd.에서 VAT에 거래의 평가를 0으로 할 수 있습니다. 그러나 로컬 VAT 회계 의무가 있을 수 있습니다. 문제가 있는 경우 세금 고문에게 확인을 합니다. 자세한 내용은 [VAT ID 추가를 참조하세요.](#add-your-vat-id)

- 아일랜드에 있으며 관련 **유효한 VAT 면세 인증을 보유하고 있습니다.** Microsoft Ireland Operations Ltd.는 VAT에서 거래를 면제할 수 있습니다. 그렇지 않으면 Microsoft Ireland Operations Ltd.에서 VAT ID 제공 여부와 관계없이 현재 아일랜드 작업 VAT 세율을 적용합니다.

- **아르메니아, 말라루스, 노르웨이, 터키 또는 아랍에미리트에 있습니다.** 로컬 VAT ID를 제공할 수 있습니다. 그러면 Microsoft Ireland Operations Ltd.에서 거래에 대한 VAT 제로 평가를 받을 수 있습니다. 하지만 현지의 VAT 회계 의무가 있을 수 있으므로 궁금한 점이 있으면 세무 담당자에게 확인하시기 바랍니다. 자세한 내용은 [VAT ID 추가를 참조하세요.](#add-your-vat-id)

Microsoft Corporation에서 청구를 수행하고(송장 참조) 미국, Microsoft 365, 푸에르토리코, 칠레 및 콜롬비아 외의 다른 국가에 등록한 경우 일반적으로 세금이 부과되지 않습니다.[](view-your-bill-or-invoice.md)

## <a name="add-your-vat-id"></a>VAT ID 추가

1. 관리 센터에서 **청구** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">내 상품</a> 페이지로 이동합니다.
2. **제품** 탭에서 업데이트할 구독을 선택합니다.
3. 구독 세부 정보 페이지의 서비스  사용 주소 섹션에서 서비스 사용 주소 **편집을 선택합니다.**
4. 서비스 **사용** 주소 편집 창의 **VAT** 번호 상자에 공백 없이도 VAT ID를 입력합니다.
5. **저장** 을 선택합니다.

## <a name="how-taxes-are-calculated"></a>세금 계산 방법

판매세는 단가에 대해 계산된 다음 집계됩니다.

예시:

>*(단가 X 세율) X 수량 = 총 판매세*

-또는-

>($1.29 X 0.095) X 100 = $12.25

## <a name="apply-for-tax-exempt-status"></a>면세 자격 신청
