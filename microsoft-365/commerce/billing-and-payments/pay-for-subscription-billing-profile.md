# <a name="how-to-pay-for-your-subscription-with-a-billing-profile"></a>청구 프로필을 사용하여 구독 비용을 결제하는 방법

구독을 구매할 때 청구 프로필을 사용하여 결제합니다. 청구 프로필은 특정 결제 방법에 연결되어 있으며 신용 카드 또는 직불 카드 또는 송장일 수 있지만 은행 계좌는 사용할 수 없습니다.

계정에 청구 프로필이 있는지 잘 알지 못하는 경우 [청구 프로필 이해하기](manage-billing-profiles.md)를 참조하세요. 청구 프로필이 없는 경우 구독 결제 방법을 [참조하세요.](pay-for-your-subscription.md)

## <a name="paying-with-recurring-billing-turned-on-or-off"></a>반기 청구를 켜거나 끄는 결제

기본적으로는 재발급 청구를 사용하는 모든 유료 구독에 대해 자동으로 재발급 청구가 설정됩니다. 청구 기간마다 해당 청구 프로필을 사용하는 모든 구독에 대해 결제하기 위해 청구 프로필과 연결된 결제 방법을 자동으로 청구합니다. 결제 방법이 거부된 경우 송장의  지금 결제 단추를 사용하여 구독에 대해 일회성 결제를 할 수 있습니다.

청구 프로필에 대해 재발급 청구가 꺼져 있는  경우 청구 프로필과 연결된 결제 방법에 관계없이 송장의 지금 결제 단추를 사용하여 청구 기간마다 결제할 수 있습니다. 수표 또는 EFT(전자 자금 이체)로 결제할 수도 있습니다. 이 작업을 하는 방법에 대한 지침은 송장의 PDF 복사본에 포함되어 있습니다.

## <a name="paying-by-invoice"></a>송장으로 결제

송장으로 결제를 설정한 청구 프로필이 있는 경우 수표 또는 EFT를 사용하여 구독 비용을 결제할 수 있습니다. 신용 카드를 사용하여 송장의 지금 결제 단추를 사용하여 온라인 결제를 할 수도 있습니다. 

송장으로 결제하려면 다음 자격을 갖추어야 합니다.

- 양호한 상태의 고객 기록
- 특정 금액 비용을 초과하는 구독 보유(이 금액은 서비스 위치에 따라 다름)
- 신용 조회 통과

신용 확인이 필요한 경우 구독을 구입할 때 알림을 받습니다. 연락에 동의한 경우 신용 승인 신청에 대한 자세한 정보가 포함된 전자 메일을 받습니다. 신용 검사는 보통 영업일 기준 2일 이내에 완료됩니다.

청구 프로필이 송장에서 지원되면 청구서를 볼 준비가 되면 전자 메일을 받을 수 있습니다. 이 전자 메일은 청구서 사본을 포함하지 않습니다. 그러나 [전자 메일로 청구서 사본을 받도록](view-your-bill-or-invoice.md#receive-a-copy-of-your-billing-statement-in-email) 선택할 수 있습니다. 청구서에는 결제 옵션에 대한 세부 정보와 전송 위치가 포함됩니다. 청구 프로필에 PO(구매 주문) 번호를 입력하면 해당 번호가 청구서에 표시됩니다. 청구서를 확인하는 방법에 대한 자세한 내용은 [청구서 또는 송장 보기](view-your-bill-or-invoice.md)를 참조하세요.

## <a name="can-i-pay-my-invoice-online"></a>온라인으로 송장을 결제할 수 있나요?

청구 프로필에 대해 재발급 청구가 꺼져 있는 경우 신용 카드를 사용하여 온라인으로 송장을 결제할 수 있습니다. 결제를 위해 청구서의 **송장에** 지금 결제 단추를 Microsoft 365 관리 센터. 송장을 찾으면 청구서 또는 [송장 보기를 참조합니다.](view-your-bill-or-invoice.md)

## <a name="can-i-change-from-my-current-payment-method-to-paying-by-invoice"></a>내 현재 결제 방법에서 송장으로 결제로 변경할 수 있나요?

청구 프로필이 신용 카드 또는 직불 카드로 백업된 경우 결제 방법을 다른 신용 카드 또는 직불 카드로만 변경할 수 있습니다. 송장 결제로 변경할 수 없습니다.

## <a name="can-i-change-from-paying-by-invoice-to-using-a-different-payment-method"></a>송장으로 결제에서 다른 결제 방법으로 변경할 수 있나요?

청구 프로필이 송장 결제로 백업된 경우 결제 방법을 변경할 수 없습니다. 송장에서  지금 결제 단추를 사용하여 신용 카드 또는 직불 카드로 결제하거나 수표 또는 EFT로 결제할 수 있습니다.
