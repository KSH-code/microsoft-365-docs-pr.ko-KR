# <a name="switch-to-a-different-microsoft-365-for-business-plan"></a>다른 비즈니스용 Microsoft 365 요금제로 전환

비즈니스가 변경되거나 더 많은 기능이 필요한 경우 요금제 전환할 수 있습니다.  

플랜을 전환하는 가장 쉬운 방법은 관리 센터에서 **플랜 전환** 단추를 사용하는 것입니다. 그러나 요금제 전환 **단추를** 사용하는 것은 모든 상황에서 지원되지 않습니다. 경우에 따라 요금제 전환을 수동으로 할 수 있습니다.


**다른 것이 필요하세요?**
- [구독 취소](cancel-your-subscription.md)
- [Microsoft 365 패밀리를 비즈니스 구독으로 업그레이드](https://support.microsoft.com/office/9322ffb8-a35d-4407-8ebe-ed6ea0859b9f.aspx)
- [Microsoft 지원 문의](../../admin/contact-support-for-business-products.md)

## <a name="use-the-switch-plans-button"></a>요금제 전환 단추 사용

요금제 전환  단추를 사용하면 현재 요금제로 전환할 수 있는 새 요금제 구입 프로세스를 진행하게 되며, 모든 사용자에게 새 요금제의 라이선스가 자동으로 할당되며 이전 요금제가 취소됩니다.

::: moniker range="o365-worldwide"

1. 관리 센터에서 **청구** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">구독</a> 페이지로 이동합니다.

::: moniker-end

::: moniker range="o365-germany"

1. 관리 센터에서 **청구** > <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">구독</a> 페이지로 이동합니다.

::: moniker-end

::: moniker range="o365-21vianet"

1. 관리 센터에서 **청구** > <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">구독</a> 페이지로 이동합니다.

::: moniker-end

2. 요금제 **전환을** 선택하여 사용 가능한 새 계획 목록을 볼 수 있습니다.

    [요금제 전환 단추가 없습니다.](#the-switch-plans-button-isnt-there)  |  [원하는 계획이 없는 경우](#i-dont-see-the-plan-i-want)

3. 월별 새 총 비용을 표시하려면 계획을 선택합니다. 키 정보를 읽고 다음을 선택하여  체크 아웃합니다.

    [도움말을 통해 새 요금제 선택.](https://go.microsoft.com/fwlink/p/?linkid=842056)

4. 도움이 **필요한 경우** **체크** 아웃 페이지에서 지금 채팅을 선택합니다.

    체크 아웃을 마치면 Microsoft 365에서 전환을 완료하는 데 몇 분 정도 걸립니다. 새 구독을 바로 사용할 수 있습니다.

## <a name="the-switch-plans-button-isnt-there"></a>요금제 전환 단추가 없습니다.

요금제 **전환** 단추를 사용할 수 없는 경우 요금제 수동 전환을 시도하거나 [지원에](switch-plans-manually.md) [문의할 수 있습니다.](../../admin/contact-support-for-business-products.md) 자세한 내용은 요금제 전환을 할 수 없는 [이유를 참조하세요.](why-can-t-i-switch-plans.md)
  
이러한 일이 일어나는 이유는 다음과 같습니다.
  
- 두 개 이상의 Microsoft 365 요금제가 사용 중입니다. 모든 사용자가 동일한  요금제에 가입한 경우 요금제 전환 단추만 사용할 수 있습니다.

- 이미 Office 365 Enterprise E5를 사용 중이기 때문에 Office 365에서 사용할 수 있는 모든 기능이 이미 있습니다.

## <a name="i-dont-see-the-plan-i-want"></a>원하는 계획이 없는 경우

요금제 전환  단추를 사용하면 전환할 수 있는 요금제가 현재 요금제의 서비스에 따라 표시됩니다. 요금제 전환  단추를 사용하여 데이터 관련 서비스가 동일한 요금제 또는 상위 버전으로 전환할 수 있습니다. 이렇게 해야 전환 중에 해당 서비스와 관련된 데이터가 손실되지 않습니다.
  
전환하기 전에 요금제 비교는 [비즈니스용 Microsoft 365](https://go.microsoft.com/fwlink/p/?linkid=842056) 제품 비교 페이지를 참조하세요. 기술 사양은 [Microsoft 365 서비스 설명을 참조하세요.](https://go.microsoft.com/fwlink/p/?linkid=842275)
  
서비스 수가 적은 계획으로 전환하려면 요금제 수동 전환 [또는](switch-plans-manually.md) [Microsoft 지원](../../admin/contact-support-for-business-products.md) 서비스에 문의하여 도움을 줄 수 있습니다.
  
## <a name="why-some-switches-take-longer"></a>일부 스위치가 더 오래 걸릴 수 있는 이유

 **요금제 전환** 시 신용 검사: 송장으로 결제하거나 특정 비용 수준에 도달하는 경우 신용 검사가 요구될 수 있습니다. 신용 검사에는 영업일 최대 2일이 걸릴 수 있습니다. 사용자는 새 요금제로 전환할 때까지 현재 요금제에 대한 모든 권한을 하게 됩니다.
  
 **비영리** 상태 확인: 일반적으로 비영리 상태에 대한 확인 프로세스를 완료하는 데 30일이 걸립니다. 30일이 지난 경우 지원에 문의하시기 바랍니다.
