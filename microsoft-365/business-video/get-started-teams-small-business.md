# <a name="get-started-with-microsoft-teams-in-your-small-business"></a>중소기업에서 Microsoft Teams 시작

- [비즈니스용 Teams 사용(동영상)](#use-teams-for-your-business-video)
- [비즈니스용 온라인 모임 호스트(동영상)](#host-online-meetings-for-your-business-video)
- [채팅 시작 및 고정(동영상)](#start-and-pin-chats-video)
- [회사 전체 팀 만들기(동영상)](#create-a-company-wide-team-video)
- [게스트가 있는 팀 만들기(동영상)](#create-a-team-with-guests-video)

작업 방식과 생활 방식이 크게 바뀌고 많은 중소기업은 원격 작업을 [포용하고 있습니다.](https://www.microsoft.com/microsoft-teams/work-remotely?ocid=oo_support_mix_marvel_ups_support_smcteamsmb_inline) 비즈니스에서 직원들이 원격으로 작업하고 고객과 가상으로 연결할 수 있도록 지원하기 위해 작업할 때 아래에는 Microsoft Teams를 사용하여 빠르게 시작하고 실행하는 데 도움이 되는 간단한 지침이 있습니다. 

원격으로 작업하기 위해 비즈니스를 변경하기 시작하면 올바른 장소로 들어오게 될 것입니다. Microsoft 365를 사용하면 비즈니스를 가상으로 관리하고 Microsoft Teams에서 온라인 모임, 채팅, [](https://www.microsoft.com/microsoft-teams/instant-messaging?ocid=oo_support_mix_marvel_ups_support_smcteamsmb_inline)파일 공유 및 통화를 통해 직원, 파트너 및 고객과 연락할 수 있습니다.

원격으로 작업하는 데 도움이 되는 경우 다음 세 단계로 시작하는 것이 좋습니다.

## <a name="1-get-ready"></a>1. 준비 

시작하기가 쉽습니다. Microsoft 365 구독이 이미 있는 경우 모든 사용자가 Microsoft Teams를 포함하는 라이선스를 포함해야 합니다. 회사에 아직 Microsoft Teams가 없는 경우 비즈니스에 적합한 [Microsoft Teams를](https://www.microsoft.com/microsoft-teams/teams-for-work) 찾아 등록할 수 있습니다.

### <a name="get-microsoft-teams-ready-for-your-customers"></a>고객을 위한 Microsoft Teams 준비

게스트 액세스를 사용하면 조직의 팀이 Microsoft Teams의 기존 팀과 채널에 대한 액세스 권한을 부여하여 조직 외부의 사용자와 공동 작업을 할 수 있습니다. 비즈니스 또는 소비자 전자 메일 계정(예: Outlook, Gmail 등)을 사용하는 모든 사람은 팀 채팅, 모임 및 파일에 대한 모든 권한을 사용하여 Microsoft Teams에서 게스트로 참여할 수 있습니다. Microsoft Teams 관리자는 게스트가 Microsoft Teams에서 사용할 수 있는(및 사용할 수 없는) 기능을 제어합니다. 게스트 액세스 관리를 [체크 아웃합니다.](https://docs.microsoft.com/microsoftteams/set-up-guests)

## <a name="2-start-using-microsoft"></a>2. Microsoft 사용 시작 

이제 Microsoft Teams 사용을 시작해야 합니다. 웹에서 사용하거나 컴퓨터, [](https://aka.ms/installteams) 노트북 또는 모바일 장치에 앱을 설치할 수 있습니다.

### <a name="meet-from-anywhere"></a>어디에서나 만나기

Microsoft Teams를 사용하여 비디오 회의를 사용하여 실시간으로 다른 참가자와 보고 대화하는 동안 HD 비디오와 안전하게 통신할 수 있습니다. 이렇게 하여 본문 언어 및 얼굴 표정이 더 높은 공동 작업 및 이해를 촉진할 때 대화에 참여와 상호 작용 수준을 추가합니다.

### <a name="host-online-meetings-for-your-business-video"></a>비즈니스용 온라인 모임 호스트(동영상)

> [!VIDEO https://docs.microsoft.com/microsoft-365/business-video/overview-online-meetings?view=o365-worldwide]

### <a name="chat-from-anywhere"></a>어디에서나 채팅

비공개 1:1 채팅, 그룹 채팅 또는 전체 팀에 표시되는 채널 대화에서 실시간으로 연결합니다. 채팅 기록은 과거의 토론 및 결정을 기억하기 위해 시간을 내어 사용할 수 있습니다. 이렇게 하여 팀에 새로 들어오거나 대화의 컨텍스트를 다시 방문하는 현재 팀 구성원에게 많은 가치를 제공합니다.

### <a name="collaborate-with-customers-and-employees"></a>고객 및 직원과 공동 작업

Microsoft Teams는 직원과 고객과 연락할 수 있는 뛰어난 방법을 제공합니다.

조직의 모든 사람이 장치에 Microsoft Teams를 설치하도록 지원합니다. 자세한 내용은 [Microsoft Teams 다운로드를 참조하세요.](https://aka.ms/downloadteams)

Microsoft Teams를 사용하여 인스턴트 메시지를 보내고, 파일을 공유하고, 모임을 열 수 있습니다. Microsoft [Teams가 없는](https://support.microsoft.com/office/c61b4f61-ee62-4a06-8bf7-0a1cd302700a)사용자도 모임에 초대할 수 있습니다. 채팅, 문서 공유 및 통화와 실시간 공동 작업을 위해 비즈니스 외부의 사용자들을 팀의 구성원으로 초대할 수 있습니다.

### <a name="create-and-organize-teams"></a>팀 만들기 및 구성

Microsoft Teams의 팀은 작업을 완료하기 위해 긴밀하게 작업하는 사용자 그룹을 모아 두기 위해 고안된 것입니다. Teams는 프로젝트 기반 작업(예: 제품 출시) 또는 회사의 부서 또는 사무실 위치에 따라 진행 중일 수 있습니다. 조직의 모든 사람을 포함하기 위해 조직 전체 팀을 설정할 수도 있습니다.

채널은 특정 항목, 프로젝트 또는 분야별로 대화를 구성하기 위한 팀 내의 전용 섹션입니다. 각 채널의 대화, 파일 및 메모는 해당 그룹의 구성원에게만 표시됩니다.

### <a name="store-and-share-files"></a>파일 저장 및 공유

팀이 원격으로 함께 작업할 때 회사 내부 및 외부에서 공유하고 공동 작업할 파일이 있습니다. Microsoft Teams를 사용하면 파일을 쉽게 공유하고 함께 작업할 수 있습니다. 문서가 Word, Excel 또는 PowerPoint 파일인 경우 동료와 고객은 Microsoft Teams 내에서 바로 문서를 보고 편집하고 공동 작업할 수 있습니다.

회사의 주요 문서를 클라우드에 저장하고 클라이언트, 고객, 파트너, 공급 업체 및 비즈니스의 다른 사용자와 안전하게 공유할 수 있도록 합니다. 또한 클라우드에 파일을 보관하면 백업의 이점과 하드 드라이브 오류, 랜섬웨어 공격 및 기타 불일치로부터 복구할 수 있는 이점을 얻을 수 있습니다.

- [비즈니스 파일 공유](https://support.microsoft.com/office/share-your-business-files-overview-6725104a-6df7-4778-99c4-c06217dffecc)

- [파일 저장 위치](https://support.microsoft.com/office/where-to-store-files-c7c20284-bc94-47f4-9728-d28e9daf0790)

## <a name="3-get-everyone-ready"></a>3. 모든 사람 준비

직원이 Microsoft Teams를 활용하도록 지원하기 위해 최종 사용자 교육 리소스에 대한 다음 링크를 공유할 수 있습니다.

- Microsoft Teams에 [](https://support.microsoft.com/office/microsoft-teams-video-training-4f108e54-240b-4351-8084-b1089f0d21d7) 대한 일련의 짧은 빠른 시작 비디오를 시청하세요.

- 모임 [사용 방법을 알아보는](https://support.microsoft.com/office/b6caa0d4-eecb-4aa0-bc75-c05f4cd907b2) 모임 비디오를 시청하세요.

- 전사적 [팀을](https://support.microsoft.com/office/create-an-org-wide-team-037bb27a-bcc9-48fe-8d72-44d9482420a3) 만들어 학습하고 작업 수행을 할 수 있는 장소를 만들거나 승격합니다.

### <a name="secure-your-business-as-it-runs-remotely"></a>원격으로 실행되는 비즈니스 보안

권장되는 보안 기능은 MFA(다단계 인증)입니다. 원격 직원에 대해 MFA를 추가하려면 Microsoft 365에 로그인할 때 텍스트, 전화 통화 또는 인증 앱에서 받은 보안 코드를 입력해야 합니다. 해커가 피싱 공격을 통해 누군가의 암호를 보유한 경우 이를 사용하여 회사의 계정 및 데이터에 액세스할 수 없습니다. Microsoft 365 관리 센터: 모바일 장치에 [대한 MFA 설정에서 MFA 보호를 추가할 수 있습니다.](https://support.microsoft.com/office/set-up-multi-factor-authentication-in-microsoft-365-business-a32541df-079c-420d-9395-9d59354f7225)

### <a name="microsoft-teams-templates"></a>Microsoft Teams 서식 파일

조직에서 배포를 보다 쉽고 간소화하기 위해 중소기업에 사용할 수 있는 서식 파일을 확인해 보아야 합니다. Microsoft Teams 템플릿을 사용하면 설정, 채널 및 미리 설치된 앱의 미리 정의된 템플릿을 제공하여 사용자가 Microsoft Teams를 효과적으로 사용할 수 있도록 하면서 팀을 빠르고 쉽게 만들 수 있습니다. 중소기업용 Teams 템플릿 시작을 [참조합니다.](https://docs.microsoft.com/microsoftteams/smb-templates)

### <a name="business-phone-system"></a>비즈니스 전화 시스템

구독에 [Microsoft 365 Business Voice를](https://aka.ms/getbusinessvoice) 추가하여 PC, Mac 또는 모바일 장치를 업무용 전화로 전환합니다. 직원에게 전화를 걸고 받을 수 있는 회사 전화 번호를 직접 제공하고, 통화를 라우팅할 통화 메뉴 시스템을 설정하고, 오디오 회의를 추가하고, 회사에 대해 무료 전화 번호와 무료 번호를 모두 설정합니다. 비용이 많이 들고 관리하기 어려운 기존 전화 통신 공급자 및 사 내 전화 시스템을 대체하면서 이 작업을 모두 합니다.
