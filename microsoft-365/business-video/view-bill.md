# <a name="view-download-or-print-your-bill-for-microsoft-365"></a>청구서 보기, 다운로드 또는 Microsoft 365

## <a name="watch-view-download-or-print-your-bill"></a>시청: 청구서 보기, 다운로드 또는 인쇄

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FGmo?autoplay=false]

비즈니스 Microsoft 365 월별 구독 청구서를 검토하고 다운로드하여 저장하거나 인쇄할 수 있습니다.

## <a name="try-it"></a>사용해 보세요!

1. In the Microsoft 365 관리 센터, select **Billing** in the left navigation pane, and then select the <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank"> **Invoices** tab</a>.
1. 송장을 선택하여 볼 수 있습니다. 송장이 없는 경우 드롭다운 목록에서 다음으로 필터링: 지난 **6개월을** 선택합니다.
1. 송장 세부 정보를 확인하려면 **송장의 오른쪽** 위 모서리에 있는 PDF 다운로드를 선택합니다.
1. PDF를 검토한 후 다운로드하여 저장하거나 인쇄할 수 있습니다. 화면의 오른쪽 위 모서리에서 다운로드 아이콘 또는 인쇄 아이콘을 선택합니다.
