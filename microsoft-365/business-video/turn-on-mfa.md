# <a name="turn-on-multifactor-authentication"></a>다단계 인증 켜기

## <a name="watch-turn-on-multifactor-authentication"></a>감시: 다단계 인증 켜기

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2MuO3?autoplay=false]

다단계 인증은 사용자와 직원이 비즈니스를 보호하는 가장 쉬운 방법 중 하나인 Microsoft 365 로그인하는 두 가지 방법을 제공해야 하다는 것을 뜻합니다.

## <a name="try-it"></a>사용해 보세요!

1. [https://admin.microsoft.com](https://admin.microsoft.com)의 관리 센터로 이동합니다.
1. 모두 **표시를 선택한** 다음 관리 Azure Active Directory **선택합니다.**
1. 를 **Azure Active Directory** **속성**, 보안 기본값 **관리를 선택합니다.**
1. 보안 **기본값 사용에서** **예를 선택한** 다음 **저장을 선택합니다.**

> [!NOTE]
> 새 구독의 경우 보안 기본값이 자동으로 설정됩니다.

선택한 직원이 다음에 로그인하면 휴대폰에서 두 번째 인증을 위해 Microsoft Authenticator 앱을 설정하도록 요청됩니다.
