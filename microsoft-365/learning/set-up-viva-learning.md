# <a name="set-up-microsoft-viva-learning-in-the-teams-admin-center"></a>Microsoft Viva Learning 센터에서 Teams 설정

Viva Learning 조직의 모든 사용자에 대해 Microsoft Teams 사용하도록 설정되어 있습니다. Viva 앱을 끄거나 끄거나 Learning 관리 센터의  앱 관리 페이지에서 조직 수준에서 viva Microsoft Teams 있습니다. 자세한 내용은 관리 센터에서 앱 Microsoft Teams [참조하세요.](/microsoftteams/manage-apps)

조직의 특정 사용자가 Viva 앱을 사용하지 Learning 차단하려면 Learning 관리 센터의 앱 관리 페이지에서 조직에  대해 Viva Microsoft Teams 설정되어 있는지 확인하십시오. 그런 다음 사용자 지정 앱 권한 정책을 만들어 해당 사용자에게 할당합니다. 자세한 내용은 Manage [app permission policies in Teams.](/microsoftteams/teams-app-permission-policies)
