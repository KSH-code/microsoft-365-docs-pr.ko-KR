# <a name="internet-sites-in-microsoft-azure-using-sharepoint-server-2013"></a>SharePoint Server 2013을 사용하는 Microsoft Azure의 인터넷 사이트

 SharePoint Server 2013을 사용하는 인터넷 사이트는 Azure 인프라 서비스에서 호스팅되는 이점을 제공합니다. 이 문서에서는 이 솔루션을 디자인하고 구현하기 위한 리소스를 제공합니다.

## <a name="using-azure-infrastructure-services-for-internet-sites"></a>인터넷 사이트에 Azure 인프라 서비스 사용

Microsoft Azure는 SharePoint Server 2013을 기반으로 하는 인터넷 사이트를 호스팅하기 위한 좋은 옵션을 제공합니다. 장점은 다음과 같습니다.

- 인프라를 구축하는 대신 멋진 사이트를 개발하는 데 집중합니다.

- 수출 및 확장을 통해 수요에 따라 솔루션을 확장할 수 있는 유연성.

- 필요한 리소스에 대한 비용만 지불하고 사용합니다.

- 고객 계정에 대해 Azure Active Directory를 활용합니다.

- Microsoft 365에서 현재 사용할 수 없는 기능(예: 심층 보고 및 분석)을 추가합니다.
