# <a name="use-powershell-to-create-reports-for-microsoft-365"></a>PowerShell을 사용하여 Microsoft 365 보고서 만들기

*이 문서는 Microsoft 365 Enterprise와 Office 365 Enterprise에 모두 적용됩니다.*

Microsoft 365 관리자 센터에서 다양한 보고서를 사용할 수 있습니다. 그러나 이러한 보고서는 너무 많은 정보를 제공하며 때로는 더 많은 정보가 필요합니다. Microsoft 365 위해 PowerShell이 필요할 때입니다.
  
이 문서에서는 Microsoft 365 PowerShell을 사용하여 Microsoft 365 테넌트의 정보를 얻는 방법에 대해 설명합니다.
  
- Microsoft 365 위해 PowerShell을 사용하여 보고부터 시작하십시오.
    
  - [Microsoft 365 용 PowerShell을 사용해야 하는 이유](./why-you-need-to-use-microsoft-365-powershell.md)
    
    
- 사용자 계정 및 라이선스에 대한 보고서:
    
  - [PowerShell을 통해 Microsoft 365 라이선스 및 서비스 보기](view-licenses-and-services-with-microsoft-365-powershell.md)
    
  - [PowerShell을 통해 라이선스 및 무면허 사용자 Microsoft 365 보기](view-licensed-and-unlicensed-users-with-microsoft-365-powershell.md)
    
  - [PowerShell을 통해 계정 라이선스 및 서비스 세부 정보를 Microsoft 365 보기](view-account-license-and-service-details-with-microsoft-365-powershell.md)
