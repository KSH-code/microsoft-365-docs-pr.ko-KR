# <a name="client-and-server-software-roadmap-for-microsoft-365"></a>클라이언트 및 서버 소프트웨어 로드맵을 Microsoft 365

대부분의 엔터프라이즈 조직에는 여러 릴리스의 운영 체제, 클라이언트 소프트웨어 및 서버 소프트웨어가 포함된 다른 환경이 있습니다. Microsoft 365 Enterprise 인프라의 주요 구성 요소 중 가장 안전한 버전이 포함되어 있습니다. 클라우드 기술을 활용하도록 설계된 생산성 기능도 포함되어 있습니다.

통합된 제품군에 대한 Microsoft 365 Enterprise 비즈니스 가치를 극대화하려면 다음의 릴리스를 마이그레이션하는 전략 계획 및 구현을 시작해야 합니다.

- 컴퓨터에 Office 클라이언트를 설치하여 엔터프라이즈용 Microsoft 365 앱.
- 서버에 Office 서버가 해당 서버의 해당 서비스에 Microsoft 365.
- Windows 7을 Windows 8.1 디바이스에서 Windows 10 Enterprise.

>[!Note]
>*2020년 1월 14일* Windows 7에 대한 지원이 종료됩니다. 자세한 내용은 지원 종료 [세부 정보를 참조하세요.](https://support.microsoft.com/help/4057281/windows-7-support-will-end-on-january-14-2020)
>

시간이 지날수록 조직은 최신 작업 공간의 비전에 [더 가까워지고 있습니다.](https://www.microsoft.com/microsoft-365/blog/2018/04/27/making-it-simpler-with-a-modern-workplace/) 이 안전하고 통합된 환경은 조직에서 팀워크와 창의성을 발휘하는 데 도움이 될 수 있습니다. Microsoft 365 Enterprise 통해 모든 방법을 지원하고 강화할 수 있습니다.

## <a name="migration-for-office-client-products"></a>클라이언트 Office 마이그레이션

대규모 조직과 소규모 조직에서는 Word, Office 및 Office 클라이언트 제품과 같은 이전 Excel 조합을 PowerPoint. 이러한 이전 버전:

- 최신 [보안 업데이트](https://support.office.com/article/install-office-updates-2ab296f3-7f03-43a2-8e50-46de917611c5) 및 지원 수정으로 업데이트할 수 있습니다. 그러나 이 프로세스는 경우에 따라 수동으로 진행될 수 있으며 조직 전체에서 확장되지 않을 수 있습니다.
- 비즈니스를 디지털 방식으로 변환하는 데 도움이 되는 Microsoft 클라우드 기술을 사용하기 위해 최적화되지 않습니다.
- 최신 기능을 제공하지 않습니다.

Microsoft 365 대한 Enterprise 포함된 엔터프라이즈용 Microsoft 365 앱. 이 버전의 Office 클라이언트 제품은 Microsoft 365 라이선스로 Enterprise 있습니다. Microsoft 클라우드에서 설치 및 업데이트됩니다. 엔터프라이즈용 Microsoft 365 앱에는 보안 업데이트와 최신 기능이 포함되어 있습니다. 자세한 내용은 에 대한 정보를 [엔터프라이즈용 Microsoft 365 앱.](/deployoffice/about-microsoft-365-apps)

### <a name="office-2007"></a>Office 2007

Office 2007 Office 버전의 경우 지원 종료가 이미 통과되었습니다. 자세한 내용은 Office [2007 지원 종료 로드맵을 참조하세요.](/deployoffice/office-2007-end-support-roadmap)

Office 2007을 실행한 컴퓨터를 Office 2010, Office 2013 또는 Office 2016으로 업그레이드하는 대신 다음 단계를 수행하십시오.

1. 사용자에 대한 Microsoft 365 라이선스를 구입하고 할당합니다.
2. 컴퓨터에서 Office 2007을 제거합니다.
3. 개별적으로 엔터프라이즈용 Microsoft 365 앱 IT 롤아웃 중에 설치합니다. 자세한 내용은 [Microsoft 365 앱용 배포 가이드](/deployoffice/deployment-guide-microsoft-365-apps)를 참조하세요.

엔터프라이즈용 Microsoft 365 앱 자동으로 설치됩니다. 클라우드 기반 서비스를 사용하여 보안 및 생산성을 향상시킬 수 있습니다.

## <a name="migration-for-office-server-products"></a>Office 서버 제품에 대한 마이그레이션

대규모 조직과 소규모 조직 모두 이전 버전의 Office 서버 제품(예: Exchange Server 및 SharePoint 서버)을 함께 사용하는 경우가 종종 있습니다. 이러한 이전 버전:

- 최신 보안 업데이트 및 지원 수정 프로그램으로 업데이트하는 것이 좋습니다. 경우에 따라 이러한 업데이트는 매월 출시됩니다.
- 비즈니스를 디지털 방식으로 변환하는 데 도움이 되는 Microsoft 클라우드 기술을 사용하기 위해 최적화되지 않습니다.
- 새 생산성 응용 프로그램(예: 새 생산성 응용 프로그램)을 포함하지 Microsoft Teams.
- 최신 보안 기능(예: Exchange 및 Defender for Office 365.

Microsoft 365 Enterprise 웹 브라우저 및 Outlook 클라이언트와 같은 Office 서버 소프트웨어의 일부와 동일한 도구 중 일부를 사용하는 클라우드 기반 버전의 Office 서버 서비스가 포함되어 있습니다. 이러한 서비스는 보안을 위해 자동으로 업데이트됩니다. 따라서 IT 담당자는 사내 서버를 유지 관리하고 업데이트하는 데 걸리는 시간을 절약할 수 있습니다. 또한 이러한 서비스는 서버 소프트웨어에 없는 새로운 기능 Office 제공합니다.

특정 작업 부하에 대한 사용자 및 데이터를 마이그레이션하는 Microsoft 365 리소스를 사용합니다.

- [사서함을 프레미스 사서함에서 Exchange Server Exchange Online](/exchange/hybrid-deployment/move-mailboxes)
- [SharePoint Server에서 SharePoint Online으로 SharePoint 마이그레이션](/sharepointmigration/migrate-to-sharepoint-online)
- [비즈니스용 Skype Online으로 Microsoft Teams](/microsoftteams/migration-interop-guidance-for-teams-with-skype)
