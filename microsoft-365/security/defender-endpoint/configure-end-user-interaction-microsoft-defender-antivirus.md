# <a name="configure-end-user-interaction-with-microsoft-defender-antivirus"></a>사용자와의 최종 사용자 상호 작용 Microsoft Defender 바이러스 백신

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**적용 대상:**

- [엔드포인트용 Microsoft Defender](/microsoft-365/security/defender-endpoint/)

네트워크의 끝점 사용자가 네트워크의 끝점과 상호 작용하는 방법을 구성할 수 Microsoft Defender 바이러스 백신. 사용자에게 Microsoft Defender 바이러스 백신, 사용자에게 어떤 알림이 표시되고, 전역적으로 배포된 그룹 정책 설정을 로컬로 다시 구성할 수 있는지 여부를 구성할 수 있습니다.

다음 문서를 사용하여 사용자와 최종 사용자 상호 작용을 Microsoft Defender 바이러스 백신

- [끝점에 나타나는 알림 구성](configure-notifications-microsoft-defender-antivirus.md) 알림 구성 및 사용자 지정, 알림에 대한 사용자 지정 텍스트 및 수정을 위해 재부팅에 대한 알림

- 사용자가 사용자 인터페이스를 [보거나](prevent-end-user-interaction-microsoft-defender-antivirus.md) 상호 작용하지 Microsoft Defender 바이러스 백신 방지 사용자로부터 사용자 인터페이스 숨기기

- [사용자가 로컬에서 정책 설정을 수정하지 못하도록 차단](configure-local-policy-overrides-microsoft-defender-antivirus.md) 사용자가 개별 끝점에서 정책 설정을 재지정하지 못하도록 방지(또는 허용)
