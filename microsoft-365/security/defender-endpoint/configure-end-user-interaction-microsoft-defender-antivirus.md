# <a name="configure-end-user-interaction-with-microsoft-defender-antivirus"></a>Microsoft Defender 바이러스 백신과 최종 사용자 상호 작용 구성

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**적용 대상:**

- [엔드포인트용 Microsoft Defender](/microsoft-365/security/defender-endpoint/)

여기에는 Microsoft Defender 바이러스 백신 인터페이스가 표시될지 여부, 사용자에게 어떤 알림이 표시되고, 전역으로 배포된 그룹 정책 설정을 로컬로 에버러할 수 있는지 여부가 포함됩니다.
