# <a name="configure-end-user-interaction-with-microsoft-defender-antivirus"></a>Microsoft Defender 바이러스 백신 최종 사용자 상호 작용 구성

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**적용 대상:**

- [엔드포인트용 Microsoft Defender](/microsoft-365/security/defender-endpoint/)

네트워크의 끝점 사용자가 Microsoft Defender 바이러스 백신 상호 작용하는 방법을 구성할 수 있습니다. 사용자가 Microsoft Defender 바이러스 백신 인터페이스를 볼지, 표시되는 알림인지, 전역적으로 배포된 그룹 정책 설정을 로컬로 재정의할 수 있는지 여부를 구성할 수 있습니다.

다음 아티클을 사용하여 최종 사용자 상호 작용을 Microsoft Defender 바이러스 백신

- **[끝점에 나타나는 알림 구성](configure-notifications-microsoft-defender-antivirus.md)** 알림에 대한 텍스트 및 수정에 필요한 재부팅에 대한 알림을 포함하여 알림을 구성하고 사용자 지정합니다.

- **[사용자가 Microsoft Defender 바이러스 백신 사용자 인터페이스를 보거나 상호 작용하는 것을 방지합니다.](prevent-end-user-interaction-microsoft-defender-antivirus.md)** 최종 사용자로부터 **바이러스 & 위협 보호** 사용자 인터페이스를 숨깁니다.

- **[사용자가 정책 설정을 로컬로 수정하지 못하도록 합니다.](configure-local-policy-overrides-microsoft-defender-antivirus.md)** 사용자가 개별 끝점에서 정책 설정을 재정의하지 못하도록 하거나 허용합니다.
