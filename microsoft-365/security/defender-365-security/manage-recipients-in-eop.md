# <a name="manage-recipients-in-standalone-eop"></a>독립 실행형 EOP에서 받는 사람 관리

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]

**적용 대상**
-  [Exchange Online Protection 독립 실행형](exchange-online-protection-overview.md)

Exchange Online 사서함이 없는 독립 실행형 EOP(Exchange Online Protection) 조직은 다음과 같은 유형의 받는 사람을 지원합니다.

- **메일 사용자:** 메일 사용자는 독립 실행형 EOP 조직의 기본 사용자 계정 유형입니다. 메일 사용자는 EOP 조직에 로그온 자격 증명이 있지만 외부 전자 메일 주소가 있습니다(사서함은 EOP 조직 외부에 위치).

  EOP에서 메일 사용자를 관리하는 방법에 대한 자세한 내용은 [EOP에서 메일 사용자 관리](manage-mail-users-in-eop.md)를 참조하세요.

- **그룹:** 다음과 같은 유형의 그룹을 만들 수 있습니다.

  - 메일 그룹
  - 메일 사용 가능 보안 그룹

  EOP에서 그룹을 관리하는 방법에 대한 자세한 내용은 [EOP에서 그룹 관리](manage-groups-in-eop.md)를 참조하세요.
