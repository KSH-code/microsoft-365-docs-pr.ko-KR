<span data-ttu-id="35b51-140">이 예에서는 이 문서에 설명된 방법을 사용하여 Microsoft에 보고되는 전자 메일 메시지를 헌트하는 Bcc Messages Reported라는 새 메일 흐름 규칙을 만들고, Bcc 받는 사람으로 laura@contoso.com julia@contoso.com 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="35b51-140">This example creates a new mail flow rule named Bcc Messages Reported to Microsoft that looks for email messages that are reported to Microsoft by using the methods described in this article, and adds the users laura@contoso.com and julia@contoso.com as Bcc recipients.</span></span>

