# <a name="microsoft-defender-for-office-365-trial-terms--conditions"></a>Office 365용 Microsoft Defender 평가판 & 조건

Microsoft Defender for Office 365 서비스의 이 무료 평가판(" 평가판")에 [](https://www.microsoftvolumelicensing.com/DocumentSearch.aspx?Mode=3&DocumentTypeId=46) 참여하면 귀하는 충돌이 발생하면 평가판 약관이 관리해야 하는 경우 온라인 서비스 약관 및 다음 약관("평가판 약관")에 구속되는 데 동의합니다. 평가판 기간은 평가판을 활성화한 날짜로부터 90일 동안입니다. 평가 기간이 만료되거나 종료하기 전에 Microsoft Defender for Office 365 구독을 구입하지 않는 한 더 이상 (i) 계정에 입력한 평가판 기능과 관련된 모든 데이터 및 (ii) 평가판 기능을 사용하여 사용자 지정한 구성 또는 사용자 지정에 액세스할 수 없습니다. Microsoft는 사전 통지 없이 책임 없이 평가판 및/또는 본 평가판 약관을 종료하거나 수정할 수 있는 권한을 보유합니다. 모든 지역 및 국가의 고객은 평가판 제품을 사용할 수 없습니다.
