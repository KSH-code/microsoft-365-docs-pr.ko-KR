# <a name="identity-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Microsoft 365 보안 센터의 ID 모니터링 및 보고

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


조직의 ID를 모니터링하고 의심스거나 위험한 동작을 추적합니다. 보고서의 **ID** 범주에서 **다음을** 추적할 수 있습니다.

* 특이 사항이 가장 많이 감지된 사용자
* 조건부 액세스 정책에 따라 위험으로 보고된 사용자 수
* 조직의 전역 관리자 수

![보고서 페이지의 ID 범주](../../media/identities.png)

특정 검색이 있는 사용자의 경우 Microsoft Defender 보안 센터에서 특정 경고를 탐색하고 조사할 수 있습니다. 검색에는 익숙하지 않은 위치에서 로그인하는 사용자와 같은 특정 상황이 포함됩니다.

위험 이벤트의 전체 집합은 Azure Active Directory 위험 [이벤트를 참조하세요.](https://docs.microsoft.com/azure/active-directory/reports-monitoring/concept-risk-events)
