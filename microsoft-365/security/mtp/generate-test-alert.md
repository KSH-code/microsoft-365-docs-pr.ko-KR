# <a name="generate-a-test-alert-in-your-microsoft-365-defender-evaluation-lab"></a>Microsoft 365 Defender 평가 랩에서 테스트 경고 생성  

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**적용 대상:**
- Microsoft 365 Defender

Microsoft 365 Defender 평가 랩 설정 및 구성을 완료한 후 위협 공격 시뮬레이션을 실행하여 통합 보안 솔루션의 감지, 경고 상관 관계 및 자동 복구 수정 기능을 경험할 수 있습니다.  

1. 로그온 https://security.microsoft.com
2. 홈 페이지에서 시뮬레이션이라는 타일을 **찾아 봐야 합니다.**  Microsoft 365 Defender에서 테스트 인시던트 만들기 단계를 위한 안내 가이드를 검토하세요.
