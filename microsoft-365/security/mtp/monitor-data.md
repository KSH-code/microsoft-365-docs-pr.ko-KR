# <a name="data-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Microsoft 365 보안 센터의 데이터 모니터링 및 보고

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**데이터** 범주를 통해서 승인받지 않은 데이터 노출이 발생할 가능성이 있는 사용자 활동을 추적할 수 있습니다. 기존 DLP 정책 보고서와 타사 DLP 정책 일치 보고서의 재작업입니다.

다음을 알 수 있습니다.

- 클라우드 앱에서 파일을 가장 많이 공유한 사용자
- 발생한 DLP 정책의 수
- DLP 정책이 무시되거나 가양성이 보고된 건수
- Microsoft Cloud App Security를 통해 타사 클라우드 서비스에서 DLP 정책 일치가 발생한 수
