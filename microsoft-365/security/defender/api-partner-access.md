### <a name="get-an-access-token-using-powershell"></a><span data-ttu-id="a0de9-190">PowerShell을 사용하여 액세스 토큰을 얻다</span><span class="sxs-lookup"><span data-stu-id="a0de9-190">Get an access token using PowerShell</span></span>

```PowerShell
$clientId = '' # Paste your application (client) ID here
$appSecret = '' # Paste your own app secret here to test, then store it in a safe place!

$resourceAppIdUri = 'https://api.security.microsoft.com'
$oAuthUri = "https://login.windows.net/$tenantId/oauth2/token"

$authBody = [Ordered] @{
    resource = $resourceAppIdUri
    client_id = $clientId
    client_secret = $appSecret

return $token
```
