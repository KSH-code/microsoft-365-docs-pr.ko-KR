# <a name="understand-the-latest-attack-campaigns-and-techniques-with-threat-analytics"></a>위협 분석을 사용하여 최신 공격 캠페인 및 기술 이해 

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


사이버 위협은 점점 더 자주 정교해지고 있습니다. 조직은 새로운 특정 위협에 대한 보안 환경과 탄력적인 상태를 신속하게 평가하는 것이 중요합니다.

**위협 분석은** 새로운 위협(새로운 기술, 공격 캠페인 및 맬웨어 발생)이 식별되는 즉시 게시되는 대화형 보고서 집합입니다. 보고서는 환경에 대한 위협의 영향을 평가하는 데 도움을 주며, 조직의 탄력을 강화하고 특정 위협을 방지하는 권장 조치를 제공합니다.

[끝점용 Microsoft Defender의 위협 분석에 대해 자세히 알아보십시오.](/windows/security/threat-protection/microsoft-defender-atp/threat-analytics)
