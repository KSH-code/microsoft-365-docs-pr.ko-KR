# <a name="signing-up-for-office-365-with-a-personal-email-address"></a>개인 전자 메일 Office 365 등록

## <a name="when-you-skip-adding-a-custom-domain"></a>사용자 지정 도메인 추가를 건너뛰는 경우


개인 전자 메일 Office 365 등록하고 지금 사용자 지정 [](../get-help-with-domains/what-is-a-domain.md) 도메인 추가를 건너뛰면 특정 기능에 대한 액세스를 제한할 수 있습니다.

지금 사용자 지정 도메인을 추가하려면 아래 모든 프리미엄 기능에 액세스할 수 있습니다.

|**기능**|**Office 365 전자 메일로 보내기**|**Office 365 도메인 사용**|
|:-----|:-----|:-----|
|**OneDrive <sup>1, 2</sup>**| [개인 OneDrive](https://onedrive.live.com/about/plans/)| [비즈니스용 OneDrive](https://onedrive.live.com/about/en-us/business/) |
|**Office 응용 프로그램: Word, Excel, PowerPoint, OneNote, Outlook, Access(PC에만 해당)**| 예 | 예
|**비즈니스 응용 <sup>프로그램 3:</sup> Microsoft Bookings 및 MileIQ**| 아니요 | 예
|**액세스 Microsoft 365 관리 센터**| 제한된 액세스(청구, 지원 및 도메인 설정) | 예
|**사용자 추가**| 아니요 | 예
|**Office 365 서비스: Exchange Online, 비즈니스용 Skype, Microsoft Teams, SharePoint Online**| 아니요 | 예
|**보안 및 규정 준수 도구**| 아니요 | 예
