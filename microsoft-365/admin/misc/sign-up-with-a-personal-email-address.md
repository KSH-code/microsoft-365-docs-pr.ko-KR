# <a name="signing-up-for-office-365-with-a-personal-email-address"></a>개인 전자 메일 주소를 사용 하 여 Office 365에 등록

## <a name="when-you-skip-adding-a-custom-domain"></a>사용자 지정 도메인 추가를 건너뛰는 경우

  
개인 전자 메일 주소를 사용 하 여 Office 365에 등록 하 고 지금은 [사용자 지정 도메인](../get-help-with-domains/what-is-a-domain.md) 추가를 건너뛰면 특정 기능에 대 한 액세스를 제한할 수 있습니다. 
  
지금 사용자 지정 도메인을 추가 하도록 선택 하면 아래의 모든 프리미엄 기능에 액세스할 수 있습니다.
