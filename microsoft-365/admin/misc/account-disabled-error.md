# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>계정에서 사용하지 않도록 설정한 오류 웹용 Outlook

If you get the error **If you have disabled** with when you try to open 웹용 Outlook (formerly known to `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` Outlook Web App), your admin might have disabled your access to 웹용 Outlook.

관리자는 사서함에 대해 사서함을 만들거나 사용하지 않도록 설정 항목의 단계를 수행하여 Outlook Web App [수 있습니다.](/exchange/recipients-in-exchange-online/manage-user-mailboxes/enable-or-disable-outlook-web-app)
