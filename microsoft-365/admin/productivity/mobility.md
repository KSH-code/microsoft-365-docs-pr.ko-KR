# <a name="mobility--people-experiences"></a>모바일 - 사람 환경

생산성 점수는 Microsoft 365를 활용하고 이를 지원하는 기술 환경을 통해 조직의 디지털 변환 여정에 대한 인사이트를 제공합니다. 조직의 점수는 사용자 및 기술 환경 측정값을 반영하며 사용자와 비슷한 조직의 벤치마크와 비교할 수 있습니다. 모바일 범주는 사람들이 경험하는 측정값의 일부입니다. 자세한 내용은 생산성 점수 개요를 [확인하고](productivity-score.md) Microsoft의 개인 정보 [취급 방침을 참조하세요.](https://privacy.microsoft.com/privacystatement)

## <a name="prerequisites"></a>필수 구성 요소

Mobility insights를 시작하려면 조직의 사용자들이 다음에 대한 라이선스를 부여해야 합니다.

- Microsoft Teams
- Exchange Online
- Word
- Excel
- PowerPoint
- OneNote

자세한 내용은 사용자에게 라이선스 할당을 [참조하세요.](../manage/assign-licenses-to-users.md)

사용자가 지난 28일 동안 적어도 한 번 이상 위의 제품에서 활동한 후 인사이트를 볼 수 있습니다.

## <a name="why-your-organization39s-mobility-score-matters"></a>조직에서 모바일&#39;중요한 이유

조직의 생산성의 기본 기조는 사람들이 어디에 있든 유연하게 작업할 수 있는 얼마나 좋은지입니다. Microsoft 365를 사용하여 Outlook, Microsoft Teams 및 사용자와 계속 연결할 수 Yammer. 또한 모든 위치 및 플랫폼에서 Word, Excel, PowerPoint 및 OneNote를 사용하여 콘텐츠에 대해 원활하게 공동 작업을 할 수 있습니다.

## <a name="how-we-calculate-the-score"></a>점수를 계산하는 방법

이 범주에 대한 주요 메트릭이 포함된 환경의 기본 인사이트를 제공합니다. 그런 다음 아래 자세히 설명된 점수 프레임워크를 이러한 메트릭에 사용하여 조직의 점수를 계산합니다.

### <a name="primary-insight"></a>기본 인사이트

Microsoft 365를 사용하면 Microsoft Outlook, Word, Excel, PowerPoint, OneNote, Microsoft Teams, Yammer 및 비즈니스용 Skype를 비롯한 앱에서 유연하게 작업할 수 있습니다. 데스크톱, 웹 및 모바일 플랫폼에서 원활한 환경을 사용하여 어디서나 작업할 수 있습니다. 기본 인사이트는 조직의 사용자에 대해 사용하도록 설정된 제품 및 이러한 사용자 중 두 개 이상의 플랫폼에서 활성 상태인 사용자 수를 확인합니다.

:::image type="content" source="../../media/primary-mobility.png" alt-text="모바일에 대한 기본 인사이트를 보여주는 차트 - 사람들이 생산성 점수를 경험합니다.":::

1. **헤더:** Microsoft 365 앱에 액세스할 수 있는 사용자와 두 개 이상의 플랫폼에서 이러한 응용 프로그램 중 하나 이상에서 활성 상태인 사용자 비율을 보여줍니다.
2. **본문:** 여러 플랫폼에서 이러한 응용 프로그램을 사용하는 경우 효율성과 만족도를 높이는 방법에 대한 자세한 정보를 제공합니다.
3. **시각화(현재 상태):** 다음과 같이 하나 이상의 Microsoft 365(아래 목록)에 대해 데스크톱, 모바일 및 웹에서 여러 플랫폼을 사용하는 사용자 수를 보여줍니다.
    - **파란색(색)** 부분이 머리줄에 표현된 백분율을 나타내는 가로 막대입니다.
    - **막대의** 분수는 헤더의 백분율을 계산하는 데 사용되는 (숫자/분모)를 강조합니다.
        - Numerator: 지난 28일 동안 데스크톱, 웹 및 모바일에서 두 개 이상의 플랫폼에서 Microsoft Outlook, Word, Excel, PowerPoint, OneNote 및 Microsoft Teams, Yammer 및 Skype 내의 응용 프로그램을 사용하는 사용자 수입니다.
        - 분모: 지난 28일 동안 Microsoft 365 앱, Exchange, Yammer, Microsoft Teams 또는 Skype에 대해 라이선스가 부여된 사용자 수입니다.
    - 키 **메트릭에 대한** 피어 벤치마크 값도 백분율로 표시됩니다.
4. **모바일 리소스 보기:** 도움말 콘텐츠를 확인하려면 이 링크를 선택합니다.

#### <a name="trend-visualization-of-the-primary-insight"></a>기본 인사이트의 추세 시각화

:::image type="content" source="../../media/mobility-primary.png" alt-text="여러 플랫폼을 사용하는 사람 수와 시간을 비교한 차트입니다.":::

이 차트는 추세선으로, 숫자는 지난 180일 동안 앱을 사용한 사용자 수입니다. 라인 차트의 각 데이터 포인트는 지난 28일간의 활동 집계입니다. 각 데이터 지점은 x 축의 각 날짜에 대해 지난 28일 동안 두 개 이상의 플랫폼에서 응용 프로그램을 사용하는 모든 사용자 수를 제공합니다.

### <a name="scoring-framework"></a>점수 매기기 프레임워크

조직의 이동성 점수는 데스크톱, 웹 및 모바일 등 다양한 플랫폼에서 사람들이 Microsoft 365 앱(Outlook, Teams, Word, Excel, PowerPoint, OneNote, Yammer 및 Skype)을 사용하는지 여부를 조직(집계) 수준에서 측정합니다.

점수는 개별 사용자 수준에서 제공되지 않습니다.

## <a name="explore-how-your-org-works-across-platforms-and-locations"></a>여러 플랫폼 및 위치에서의 운영 방식 살펴보기

또한 조직의 사용자가 여러 플랫폼에서 작업하는 방법을 쉽게 알 수 있도록 하는 정보도 제공합니다. 이러한 추가 메트릭은 생산성 점수에 직접적인 영향을 주지 않지만 디지털 변환의 일부로 작업 계획을 세우는 데 도움이 됩니다.  

### <a name="use-of-outlook-across-platforms"></a>플랫폼 전체에서 Outlook 사용

:::image type="content" source="../../media/outlook-across-platforms.png" alt-text="여러 플랫폼에서 Outlook을 사용하는 사용자 수를 보여하는 차트입니다.":::

1. **헤더:** 여러 플랫폼에서 Outlook을 사용하는 Outlook에서 활성 상태의 비율을 보여줍니다.
2. **본문:** 모바일 장치에서 Outlook을 사용하여 전자 메일의 어느 곳에서나 연결할 수 있는 값에 대한 정보를 제공합니다.
3. **시각화:** Outlook에서 활성 상태인 하나 이상의 플랫폼을 사용하는 비율을 보여줍니다.
      - **여러 플랫폼:**
        - Numerator: 지난 28일 동안 데스크톱, 모바일 또는 웹의 두 개 이상의 플랫폼에서 Outlook을 사용한 사용자 수입니다.
        - 분노: 지난 28일 동안 Outlook을 한 번 이상 사용한 사람 수입니다.
      - **데스크톱 전용:**
        - Numerator: 지난 28일 동안 데스크톱 플랫폼에서만 Outlook을 사용한 사용자 수입니다.
        - 분노: 지난 28일 동안 Outlook을 한 번 이상 사용한 사람 수
      - **웹 전용:**
        - Numerator: 지난 28일 동안 웹 플랫폼에서만 Outlook을 사용한 사용자 수입니다.
        - 분노: 지난 28일 동안 Outlook을 한 번 이상 사용한 사람 수입니다.
      - **모바일 전용:**
        - Numerator: 지난 28일 동안 모바일 플랫폼에서만 Outlook을 사용한 사람 수입니다.
        - *분노: 지난 28일 동안 Outlook을 한 번 이상 사용한 사람 수입니다.

### <a name="use-of-teams-across-platforms"></a>플랫폼에서 Teams 사용

:::image type="content" source="../../media/teams-across-platforms.png" alt-text="하나 이상의 플랫폼에서 Teams를 사용하는 사람 수를 보여하는 차트입니다.":::

1. **헤더:** Microsoft Teams에서 활성 상태인 사용자 중 여러 플랫폼에서 사용하고 있는 비율을 보여줍니다.
2. **본문:** 모바일 장치에서 Teams를 사용하는 가치에 대한 정보를 제공하면 사용자가 모든 위치에서 작업하면서 메시지를 최신으로 유지 할 수 있습니다.
3. **시각화:** 단일 플랫폼 또는 여러 플랫폼에서 Microsoft Teams를 사용하고 있는 사용자 중 백분율을 보여줍니다. 
    - **여러 플랫폼:**
        - 숫자: 데스크톱, 모바일 또는 웹 플랫폼 중 2개 이상에서 지난 28일 동안 Teams를 사용한 사용자 수입니다.
        - 분노: 지난 28일 동안 Microsoft Teams를 한 번 이상 사용한 사용자 수입니다.
    - **데스크톱 전용:**
        - 숫자: 지난 28일 동안 데스크톱 플랫폼에서만 Microsoft Teams를 사용한 사용자 수
        - 분노: 지난 28일 동안 Teams를 한 번 이상 사용한 사람 수
    - **웹 전용:**
        - 숫자: 지난 28일 동안 웹 플랫폼에서만 Microsoft Teams를 사용한 사용자 수입니다.
        - 분노: 지난 28일 동안 Microsoft Teams를 한 번 이상 사용한 사용자 수
    - **모바일 전용:**
        - 숫자: 지난 28일 동안 모바일 플랫폼에서만 Microsoft Teams를 사용한 사용자 수
        - 분노: 지난 28일 동안 Teams를 한 번 이상 사용한 사람 수

### <a name="use-of-microsoft-365-apps-across-platforms"></a>플랫폼 전체에서 Microsoft 365 앱 사용

:::image type="content" source="../../media/microsoft365apps-platforms.png" alt-text="다중 또는 단일 플랫폼에서 Microsoft 365 앱을 사용하는 사용자 수를 표시하는 차트입니다.":::

1. **헤더:** 여러 플랫폼에서 Microsoft 365 앱(Word, Excel, PowerPoint 및 OneNote)에서 활동하는 사용자 비율을 보여야 합니다.
2. **본문:** 조직의 사용자에게 어디에서나 파일에 액세스할 수 있는 유연성을 제공하는 가치에 대한 정보를 제공합니다.
3. **시각화:** 그룹화된 세로는 단일 또는 여러 플랫폼에서 고려되는 각 앱(Word, Excel, PowerPoint 및 OneNote)을 사용하는 사용자 수를 나타내기 위한 것입니다. 이러한 각 응용 프로그램에 대해 막대는 다음을 표현합니다.
      - **여러 플랫폼:** 지난 28일 동안 두 개 이상의 플랫폼에서 앱에서 활성 상태인 사용자 수입니다.
      - **데스크톱 전용:** 지난 28일 동안 데스크톱 플랫폼에서만 앱에서 활성 상태인 사용자 수입니다.
      - **웹 전용:** 지난 28일 동안 웹 플랫폼에서만 앱에서 활성 상태인 사용자 수입니다.
      - **모바일 전용:** 지난 28일 동안 모바일 플랫폼에서만 앱에서 활성 상태인 사용자 수입니다.

### <a name="remote-work"></a>원격 작업

:::image type="content" source="../../media/remotework-trend.png" alt-text="원격, 현장 또는 둘 다에서 작업하는 인원 수를 보여하는 차트입니다.":::

1. **헤더:** 회사 네트워크 외부의 집이나 위치에서만 작업하는 비율을 보여줍니다.
2. **본문:** 조직의 실제 사무실에 액세스하지 않고도 사용자에 대한 원격 작업을 촉진하는 것이 얼마나 중요한지 강조합니다.
3. **시각화:** 원격으로만 작업하는 일별 비율 및 일일 작업 인원의 일별 백분율에 대한 추세선이 표시됩니다. 사용자가 하루 동안 Microsoft 365 앱에서 3시간 이상의 활동을 수행하는 경우 사용자가 출장으로 간주됩니다.
