# <a name="email-collaboration"></a>전자 메일 공동 작업

Microsoft 365 메일 그룹, 메일 그룹Outlook 공유 사서함 및 공용 폴더의 그룹을 통해 공동 작업을 장려합니다. 이러한 옵션은 용도, 사용자 환경, 기능 집합이 각각 다릅니다. 사용할 옵션은 사용자가 수행해야 하는 작업과 조직에서 어떤 도구를 제공하는지에 따라 달라집니다.
  
## <a name="summary-of-collaboration-options"></a>공동 작업 옵션 요약
<a name="BKMK_SUMMARYOFCOLLABORATIONOPTIONS"> </a>

이 표에서는 사용할 수 있는 다양한 공동 작업 옵션에 대해 설명하고 있습니다.
  


|**공동 작업 도구**|**설명**|
|:-----|:-----|
|그룹의 Outlook  <br/> |공유 작업 영역의 모든 응용 프로그램에서 Microsoft 365. 여기에는 공유 받은 편지함, 일정, 파일 저장을 위한 비즈니스용 OneDrive 사이트가 포함되어 있습니다. 사용자는 전자 메일 또는 일정에서 바로 Outlook 그룹을 만들고 찾고 가입할 수 있습니다. Exchange Online 또는 Microsoft 365 신규 및 기존 사용자는 Outlook.  <br/> |
|공유 사서함  <br/> |특정 사용자가 전자 메일 메시지를 읽고 보내며 공통 일정을 공유할 수 있는 사서함입니다. 공유 사서함은 고객이 회사에 대해 문의하는 데 사용할 수 있는 일반 전자 메일 주소(예: info@contoso.com 또는 sales@contoso.com) 역할도 할 수 있습니다. 공유 사서함에서 다른 이름으로 저장 권한이 설정되어 있으면 사서함에서 보내는 전자 메일에 일반 주소(예: sales@contoso.com)가 사용됩니다.  <br/> |
|메일 그룹  <br/> |두 명 이상의 사용자에게 동시에 전자 메일 메시지를 배포하는 데 사용합니다. 메일 그룹은 메일 사용이 가능한 배포 그룹이라고도 합니다. 동적 메일 그룹이라는 메일 그룹의 변형은 변화하는 다수의 받는 사람 그룹에 전자 메일을 보내기 위해 사용되는 메일 사용이 가능한 Active Directory 그룹 개체입니다. 받는 사람은 특정 로캘의 모든 구성원이나 모든 상근 직원처럼 사용자가 지정한 필터와 조건에 의해 결정됩니다.<br/><br/> Microsoft 365 그룹의 Outlook 메일 그룹보다 공동 작업을 위한 더 강력한 솔루션을 제공합니다. 자세한 내용은 [메일](https://support.microsoft.com/office/7fb3d880-593b-4909-aafa-950dd50ce188) 그룹을 그룹의 그룹으로 업그레이드해야 하는 이유 및 Outlook 메일 그룹을 Microsoft 365 [참조하세요.](../manage/upgrade-distribution-lists.md)  <br/> |
|공용 폴더  <br/> |공유 액세스를 위해 설계된 공용 폴더는 조직의 다른 사용자와 정보를 수집, 구성 및 공유할 수 있는 쉽고 효과적인 방법을 제공합니다. 공용 폴더는 쉽게 찾을 수 있고 Outlook 폴더 보기에 항상 표시되는 상세한 계층 구조로 콘텐츠를 구성합니다. 공용 폴더는 메일을 사용하도록 설정할 수 있으며 메일 그룹의 구성원으로 추가할 수 있습니다. 메일 그룹으로 보낸 전자 메일은 보관하거나 나중에 참조할 수 있도록 공용 폴더에 자동으로 추가됩니다. 또한 공용 폴더는 SharePoint Online 구독이 없는 경우에 간단하게 문서를 공유할 수 있는 방법을 제공합니다.  <br/> |
