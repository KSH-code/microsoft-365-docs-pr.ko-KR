# <a name="install-office-applications"></a>Office 응용 프로그램 설치

이제 설정한 Microsoft 365 Mac, PC 또는 모바일 Office 개별 응용 프로그램을 설치할 수 있습니다.
  
방법에 대한 자세한 내용은 다음 링크를 참조하세요.
  
- Office 응용 프로그램 설치: PC [Office Mac에](https://support.microsoft.com/office/4414eaaf-0478-48be-9c42-23adc4716658) 설치

- 다른 앱 [설치:](https://support.microsoft.com/office/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281)Project [,](https://support.microsoft.com/office/install-visio-f98f21e3-aa02-4827-9167-ddab5b025710)Visio [또는](https://support.microsoft.com/office/install-skype-for-business-8a0d4da8-9d58-44f9-9759-5c8f340cb3fb) 비즈니스용 Skype

- 모바일 장치 설정: [Microsoft 365 설정 - 도움말](https://support.microsoft.com/office/7dabb6cb-0046-40b6-81fe-767e0b1f014f)

- 전자 메일에서 전자 메일 [Outlook:](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b) Windows [또는 Mac](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b#PickTab=Outlook_for_Mac)
 
- [사용자를 최신 앱으로 업그레이드합니다.](upgrade-users-to-latest-office-client.md) 

::: moniker range="o365-21vianet"

AADP(Azure Active Directory Premium) 플랜 1 또는 플랜 2를 구입한 경우 Microsoft Identity Manager(MIM). 다운로드 MIM 다운로드 [센터로 이동 합니다.](https://www.microsoft.com/zh-cn/download/details.aspx?id=58498)
