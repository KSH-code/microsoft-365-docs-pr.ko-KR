# <a name="customize-the-microsoft-365-theme-for-your-organization"></a>조직의 Microsoft 365 테마 사용자 지정

조직의 관리자는 조직의 사용자에 대해 여러 테마를 만들고 조직의 여러 구성원에게 적용할 테마를 선택할 수 있습니다. 조직 테마는 조직의 사용자에 대한 위쪽 탐색 모음에 표시됩니다.

You can add or update a default theme that applies to everyone within your org. 또한 여러 Microsoft 365 그룹에 할당할 수 있는 최대 4개의 추가 그룹 테마를 만들 수 있습니다.
  
## <a name="add-or-update-your-organizations-theme"></a>조직의 테마 추가 또는 업데이트

1. 관리 센터에서 설정 조직  설정 페이지로 이동한 다음 조직 \>  프로필 **탭을** 선택합니다.

2. 조직 **프로필 탭에서** 조직 테마 **를 선택합니다.**

다음 탭을 사용하여 모든 테마를 사용자 정의할 수 있습니다.

|**Tab**|**어떻게 해야 하나요?**|
|:-----|:-----|
|[일반](#general-modify-a-theme) <br/> |테마 이름을 수정하고 최대 5개 그룹에 할당합니다(해당하는 경우).  <br/> |
|[로고](#logos-specify-your-theme-logos) <br/> |Office 어두운 테마 및 모바일 옵션을 포함하여 테마 로고를 추가합니다.  <br/> |
|[색](#colors-choose-theme-colors) <br/> |탐색 모음, 강조색, 텍스트 및 아이콘 색을 지정하여 색 구성표를 사용자 지정합니다. <br/> |

## <a name="general-modify-a-theme"></a>일반: 테마 수정

일반 탭의 환경은 기본 테마를 추가하거나 수정하는지 또는 그룹 테마를 수정하는지 여부에 따라 결정됩니다.

### <a name="update-the-default-theme"></a>기본 테마 업데이트

기본 테마는 표시되는 첫 번째 테마입니다.  

1. 이전에 조직에 대한 테마를 사용자  지정한 경우 기본 테마를 선택하고 저장된 사용자 지정 중 하나를 사용하거나 테마 추가 **를 선택합니다.**
2. 일반 **페이지에서** 사용자가 테마를 다시 지정하지 못하도록 방지하고 사용자의 표시 이름을 표시할 수 있습니다.
3. **저장** 을 선택하여 변경 내용을 저장합니다.  

> [!IMPORTANT]
> 기본 테마는 고유하며 이름을 변경할 수 없습니다. 조직 내의 모든 사용자에 적용됩니다. 기본 테마를 삭제하려면 먼저 다른 모든 테마를 삭제해야 합니다.

:::image type="content" source="../../media/default-theme-general.png" alt-text="Screenshot: General tab showing the default theme for your organization":::

### <a name="create-a-group-theme"></a>그룹 테마 만들기

최대 4개의 추가 그룹 테마를 만들 수 있습니다.

1. 일반 **페이지에서** 테마의 이름을 입력합니다.

2. **그룹에서** 기본 테마를 사용하는 대신 그룹 테마를 볼 수 있는 최대 5개 Microsoft 365 그룹을 선택할 수 있습니다. 또한 사용자가 테마를 다시 지정하지 못하도록 방지하고 사용자의 표시 이름을 표시할 수 있습니다.

3. **저장** 을 선택합니다.

:::image type="content" source="../../media/default-theme-general-users.png" alt-text="Screenshot: General tab showing the default theme for a group of users in your organization":::

## <a name="logos-specify-your-theme-logos"></a>로고: 테마 로고 지정

로고 **페이지에서** 로고를 추가하고 사용자가 로고를 선택할 때 이동할 URL을 지정할 수 있습니다.

- **기본 로고:** 로고를 지정하는 URL 위치를 추가합니다. URL에서 HTTPS를 사용하며 이미지가 200 x 30 픽셀 이상인지 확인합니다. 기본 로고는 JPG, PNG, GIF 또는 SVG 형식일 수 있습니다.
- **대체 로고:** 로고를 지정하는 URL 위치를 추가합니다. 대체 로고는 Office 어두운 테마에서 사용하기 위해 최적화해야 합니다. 기본 로고와 동일한 요구 사항
- **작은 기본 로고:** 로고를 지정하는 URL 위치를 추가합니다. 이미지는 48 x 48픽셀 이상입니다. 소형 또는 모바일 장치에 맞게 이 이미지를 확장할 수 있습니다.
- **작은 대체 로고:** 로고를 지정하는 URL 위치를 추가합니다. 이 이미지의 요구 사항은 작은 기본 로고와 동일합니다.
- **On-click link:** Add a URL location that points to your logo. 로고를 회사 웹 사이트와 같은 회사 리소스에 대한 링크로 사용할 수 있습니다.

**저장** 을 선택하여 변경 내용을 저장합니다.

로고는 어떤 경우든 제거할 수 있습니다. 로고 **페이지로 돌아가서** 제거를 **선택합니다.**

:::image type="content" source="../../media/default-theme-logos.png" alt-text="Screenshot: Logos tab showing default theme logo for your organization":::

> [!NOTE]
> 기본적으로 대부분의 조직에서 사용하는 로고 선택이 먼저 표시됩니다. 모든 로고 선택을 표시하려면 목록 맨 아래로 이동하여 고급 옵션 **참조를 선택합니다.**
  
## <a name="colors-choose-theme-colors"></a>색: 테마 색 선택

색 **페이지에서** 기본 색을 설정하고 사용할 로고를 선택할 수 있습니다.

- **탐색 모음 색:** 탐색 모음의 배경에 사용할 색을 선택합니다. 탐색 모음이 모든 페이지의 맨 위에 표시됩니다.
- **텍스트 및 아이콘 색:** 위쪽 탐색 모음의 텍스트 및 아이콘에 사용할 색을 선택합니다.
- **강조색:** 흰색 또는 밝은 배경에 잘 표시될 색을 선택 합니다. 강조색은 흰색 또는 밝은 배경에 일부 링크와 단추의 색을 표시하는 데 사용됩니다. 예를 들어 악센트 색은 사용자의 받은 편지함 및 해당 포털 페이지의 요소에 Office.com 사용됩니다.
- **색 다시 설정:** 이 링크를 선택하여 색을 기본 색으로 다시 설정합니다.
- **어떤 로고를 사용해야 하나요?**: 직접 만든 기본 로고 또는 기타 로고를 선택합니다.

:::image type="content" source="../../media/default-theme-colors.png" alt-text="Screenshot: Colors tab showing default theme colors for your organization":::

## <a name="frequently-asked-questions"></a>질문과 대답

### <a name="my-organization-already-has-a-theme-for-all-employees-how-will-this-change"></a>조직에 이미 모든 직원에 대한 테마가 있습니다. 이 변경은 어떻게 될까요?

기본 테마는 모든 직원에게 계속 표시됩니다.새 그룹 테마 추가는 해당 테마와 연결된 Microsoft 365 그룹에서만 사용할 수 있습니다.

### <a name="whydont-isee-group-themes-in-the-admin-center"></a>관리 센터에 그룹 테마가 없는 이유는 무엇입니까?

전역 관리자만 회사 테마를 사용자 지정할 수 있습니다.전역 읽기 전용 권한이 있습니다.

### <a name="how-many-different-themes-can-i-set-up-for-my-organization"></a>조직에 대해 설정할 수 있는 테마는 몇 개입니까?  

최대 5개의 테마를 만들 수 있습니다. 기본 테마 및 4개의 그룹 테마.  

### <a name="can-i-use-security-groups-or-distribution-groups-instead-of-microsoft-365-groups"></a>Microsoft 365 그룹 대신 보안 그룹 또는 메일 그룹을 사용할 수 있나요?

아니요. 새 그룹 테마는 보안 그룹이나 메일 그룹이 아닌 하나 이상의 Microsoft 365 그룹에 매핑해야 합니다.

> [!NOTE]
> Outlook에서 메일 [그룹을 Microsoft 365 그룹으로 변환할](../manage/upgrade-distribution-lists.md) 수 있습니다.

### <a name="can-imanually-assign-a-theme-independent-ofmicrosoft-365-groups"></a>Microsoft 365 그룹과 독립적인 테마를 수동으로 할당할 수 있나요?  

아니요. 새 그룹 테마는 하나 이상의 Microsoft 365 그룹에 매핑해야 합니다. Microsoft 365 그룹의 구성원인 사용자에게는 해당 그룹에 테마가 적용됩니다. 관리 센터의 설정 그룹으로 이동하여 새 구성원을 만들고 **** [Microsoft 365](../create-groups/create-groups.md) 그룹에 추가할   >  **** 수 있습니다.

### <a name="what-happens-if-a-user-is-assigned-to-multiple-group-themes"></a>사용자가 여러 그룹 테마에 할당된 경우 어떻게 하나요?  

여러 그룹 테마에 할당된 사용자에게 기본 테마가 표시됩니다.  

### <a name="why-cant-i-delete-the-default-theme"></a>기본 테마를 삭제할 수 없는 이유는 무엇입니까?  

기본 테마는 모든 그룹 테마가 삭제된 후만 삭제할 수 있습니다. 그룹 테마를 삭제하기 전에 모든 그룹 테마를 삭제해야 합니다.

### <a name="why-am-i-receiving-an-error-message-every-time-i-uploadalogo-url"></a>로고 URL을 업로드할 때마다 오류 메시지가 수신되는 이유입니다.  

사용하는 로고가 공개 주소 지정 가능 URL로 지정되어 있는지 확인합니다. SharePoint Online을 사용하여 [Azure Blob Storage](/azure/storage/blobs/storage-upload-process-images?tabs=dotnet) 또는 Office [365 콘텐츠](../../enterprise/use-microsoft-365-cdn-with-spo.md)배달 네트워크에 로고를 업로드하려면 다음 단계를 따르세요.

### <a name="why-am-i-receiving-themessagedoesnt-meet-minimum-color-contrast-ratio-of-451"></a>"4.5:1의 최소 색 명암비를 충족하지 않습니다."라는 메시지가 수신된 이유는 무엇입니까?

텍스트, 아이콘 또는 단추 색과 배경색 간의 권장 명암비는 4.5:1입니다. 이 권장 사항을 다시 설정하고 이 요구 사항이 아니기 때문에 테마를 저장할 수 있습니다.

### <a name="if-i-define-a-theme-which-places-in-microsoft-365-will-this-be-used"></a>테마를 정의하는 경우 Microsoft 365에서 이 테마를 사용할 위치는 무엇입니까?

모든 테마는 Microsoft 365 제품군 헤더의 일부로 조직의 모든 사용자에 대한 위쪽 탐색 모음에 표시됩니다.
