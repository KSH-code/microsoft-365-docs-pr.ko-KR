# <a name="change-your-organizations-address-technical-contact-and-more"></a>조직 주소, 기술 담당자 등 변경
  
조직 이름, 주소, 전화 및 기술 연락처와 같은 조직 프로필을 변경할 수 있습니다. **이 정보를 업데이트하려면 전역 관리자여야 합니다.**
  
청구서 또는 구독과 연결된 주소를 변경하려면 [비즈니스 Microsoft 365 대한 청구 주소 변경을](../../commerce/billing-and-payments/change-your-billing-addresses.md)참조하십시오.

## <a name="edit-organization-information"></a>조직 정보 편집

> [!IMPORTANT]
> 구독의 국가 또는 지역을 변경할 수 없습니다. 조직에 본사를 둔 국가 또는 지역이 귀하가 사용할 수 있는 서비스, 세금 및 청구 통화 및 데이터 센터의 위치를 결정하기 때문입니다. 조직의 국가 또는 지역을 변경하려면 새 계정에 가입하고 원하는 국가 또는 지역을 선택하고 새 구독을 구입합니다.

회사의 프로필 페이지의 다른 정보를 변경하려면 다음을 수행하십시오.
  
1. 관리 센터에서 설정 Org  \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2053743" target="_blank">**설정**</a> 페이지로 이동합니다.

2. 조직 **프로필** 탭에서 **조직 정보를** 선택합니다.

3. 조직의 정보를 업데이트한 다음 **변경 내용 저장을** 선택합니다. 변경 내용을 저장할 수 있도록 *로 표시된 모든 필수 필드를 작성해야 합니다.

각 필드에 대한 설명은 다음과 같습니다.

## <a name="what-do-these-fields-mean"></a>이러한 필드는 무엇을 의미하나요?

|**필드**  |**설명**  |
|---------|---------|
|이름  <br/>   | 여기에 입력된 이름은 다음 페이지에서 사용자에게 표시됩니다.  <br/>  로그인 페이지: 사용자가 회사 또는 학교 전자 메일 주소를 사용하여 다른 Microsoft 계정을 설정했을 경우 로그인 페이지에 조직 이름이 표시될 수 있습니다. 그러면 회사 또는 학교 계정과 다른 계정을 손쉽게 구분할 수 있으므로 로그인할 때 사용할 계정을 식별할 수 있습니다.  <br/>  조직 프로필 링크 및 페이지: 조직 프로필의 링크에 조직 이름이 표시됩니다.  <br/>  Yammer 탐색: Yammer 왼쪽 탐색에서는 조직 이름을 홈 Yammer 네트워크의 이름으로 사용합니다.  <br/> 동기화 클라이언트 OneDrive: 조직 이름은 Windows 파일 탐색기및 Mac의 파인더, 파일 경로, OneDrive 활동 센터, OneDrive 클라우드 아이콘의 도구 설명 및 OneDrive 설정 창에 표시됩니다. 현재 조직 이름을 업데이트해도 구성된 클라이언트에 대해 업데이트되지 않습니다. <br/> MS Teams: Teams 조직 스위처가 조직 이름을 표시합니다. <br/>  |
|주소, 구/군/시, 시/도, 우편 번호  <br/>     | 여기에 입력된 주소는 청구서에 표시되는 주소 [Microsoft 365입니다.](../../commerce/billing-and-payments/understand-your-invoice2.md)  <br/>        |
|국가 또는 지역  <br/>    | 회사의 본사가 있는 국가 또는 지역입니다. 선택한 국가 또는 지역은 사용 가능한 서비스, 해당 국가 또는 지역의 세금 및 대금 청구 통화, 가장 가까운 데이터 센터의 위치를 결정합니다( [Microsoft Office 라이선스 제한 사항](https://office.microsoft.com/redir/FX103037529) 참조).  <br/>참고: 일단 선택되면 국가 또는 지역을 변경할 수 없습니다. 선택을 변경하려면 구독을 취소하고 다시 등록해야 합니다. 이 프로세스에 대한 도움말을 보려면 [지원팀에 문의하십시오.](../../business-video/get-help-support.md)        |
|전화  <br/>     | 회사의 기본 전화입니다. 일반적으로 회사 본사의 전화 번호를 말합니다.  <br/>        |
|기술 담당자  <br/> |Microsoft 365 구독을 관리하는 기본 기술 담당자의 이메일 주소입니다. Microsoft 365 서비스 상태에 대한 통신을 받게 되는 사람입니다.  <br/> |
|기본 설정 언어  <br/> |기본 설정 언어에 따라 Microsoft에서 조직으로 보내는 모든 통신의 언어가 결정됩니다. 등록할 때 이 설정에 따라 SharePoint Online에서 사용되는 언어가 결정되며 사용자의 팀 사이트도 이 언어로 표시됩니다. 등록한 후 언어 기본 설정을 변경할 경우 이후 진행되는 모든 통신에는 최근에 선택한 언어가 사용됩니다.    <br/> 참고: SharePoint 온라인에서 사용하는 언어는 변경할 수 없습니다.           |

## <a name="change-your-alternate-email-address"></a>대체 이메일 주소 변경

관리자는 암호를 재설정하기 위해 대체 이메일 주소를 변경할 수 있습니다.

1. <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">Microsoft 365 관리 센터로</a>이동합니다.

2. 헤더에서 프로필 아이콘을 선택한 다음 **계정 보기를** 선택합니다.

3. 왼쪽에서 보안 **정보를 선택합니다.**

4. 이메일 섹션에서 변경 **선택**

5. **대체 이메일** 상자에서 주소를 편집한 다음 **다음을** 선택합니다.

6. 대체 이메일 주소로 전송된 코드를 입력하고 **확인을** 선택하여 변경 여부를 확인합니다.
다른 프로필 정보 변경에 대해 알아보려면 [연락처 기본 설정 변경](change-contact-preferences.md) 또는 표시 언어 [변경을](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b.aspx)참조하십시오.
  
### <a name="email-signatures"></a>전자 메일 서명
  
Outlook Web App에서 전자 메일 서명을 변경할 수 있습니다. 자세한 내용은 [메일 설정](https://support.microsoft.com/office/30c69a79-efc6-42d2-b740-4bf1c1f8a01c)을 참조하세요.
