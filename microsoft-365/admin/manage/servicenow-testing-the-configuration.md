# <a name="testing-the-servicenow-configuration"></a>ServiceNow 구성 테스트

응용 프로그램에 외부 시스템과의 성공적인 통신이 필요한 경우 성공적인 구성을 위해 연결을 테스트하는 방법을 간략하게 설명하십시오.

다음 단계에 따라 지원 통합을 Microsoft 365 테스트합니다.

1. ServiceNow 포털에 관리자로 로그인합니다.

2. 인시던트 열기

3. Microsoft 365 **지원** 탭에 집중하고 Microsoft 365 Insights  해결 방법을 성공적으로 반환할지 여부를 선택합니다.
