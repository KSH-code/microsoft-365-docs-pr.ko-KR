# <a name="manage-messages-in-message-center"></a>메시지 센터에서 메시지 관리

메시지 센터 게시물은 새로운 기능과 변경된 기능에 대한 중요한 공지입니다. 이 문서에서는 조직의 변경 관리에 도움이 되는 메시지로 할 수 있는 일에 대해 보다 자세한 정보를 제공합니다.
  
## <a name="read-a-message"></a>메시지 읽기

메시지 센터로 이동하려면 관리 [](https://go.microsoft.com/fwlink/p/?linkid=2024339)센터로 이동하여 상태 메시지   >  **센터를 선택합니다.** 

메시지 제목을 선택하여 읽기 창에서 여십시오. 모든 메시지에는 변경에 대한 요약, 변경이 사용자에 미칠 수 있는 영향에 대한 설명, 조직에서 변경을 준비하는 방법에 대한 정보 및 대부분의 경우 자세한 내용을 볼 수 있는 추가 정보 링크가 포함되어 있습니다. 메시지가 만료된 후에도 추가 정보를 계속 사용할 수 있습니다.

## <a name="give-feedback-on-a-post"></a>게시물에 대한 피드백 제공

메시지 센터에서 세부 정보를 볼 메시지를 선택할 수 있습니다.

메시지에 대한 피드백을 제공하려면 Like 또는  **Dislike** 아이콘을 선택하고 나타나는 텍스트 상자에 피드백을 제공합니다. 개인 정보는 제공하지 않습니다. 원하는 경우 이  피드백에 대해 연락을 하여 보내기 를 선택할 **수 있습니다.**

## <a name="share-a-message"></a>메시지 공유

다른 사람이 행동해야 하는 메시지를 확인하나요? 전자 메일로 메시지 내용을 모든 사용자와 공유할 수 있습니다.
  
1. 메시지를 선택하여 열고 공유를 **선택합니다.**
  
2. 메시지를 공유하기 위해 최대 2개의 전자 메일 주소를 콜론으로 구분하여 입력합니다. 개별 및 그룹 전자 메일 주소로 보낼 수 있습니다. 원하는 경우 전자 메일로 메시지 복사본을 받거나(메시지가 기본 전자 메일 주소로 이동) 받는 사람에게 더 많은 컨텍스트를 제공하기 위해 개인 메시지를 추가할 수 있습니다.
  
3. 전자 **메일을 보내기** 위해 공유를 선택합니다.
    
## <a name="get-a-link"></a>링크 다운로드

다른 관리자와 함께 변경을 인식하고 조치를 취하는지 확인해야 하나요? 예를 들어 사용자를 해당 메시지에 직접 연결하는 전자 메일 또는 인스턴트 메시징에서 공유하는 링크를 생성할 수 있습니다. 링크를 공유하는 사람은 조직의 Microsoft 365 관리자 또는 연결된 메시지에 액세스할 수 없습니다.

::: moniker range="o365-worldwide"

2. **클립보드(링크)** 아이콘에 복사 링크를 선택합니다.

3. 웹 페이지가 클립보드에 액세스할 수 있도록 허용합니다. 직접 링크가 클립보드에 복사됩니다.

## <a name="read-and-unread-states"></a>읽기 및 읽지 않은 상태

읽을 수 없는 메시지 센터의 모든 메시지는 굵게 표시됩니다. 메시지를 열면 읽은 것으로 표시됩니다. 메시지를 언더로 표시하면 됩니다.

::: moniker range="o365-worldwide"

1. 메시지 센터의 기본 페이지에서 메시지를 선택합니다.

2. 메시지  맨 위에 있는 언더(봉투) 아이콘으로 표시를 선택합니다.
  
::: moniker-end

::: moniker range="o365-germany"

1. 메시지를 선택하여 열습니다.

2. **언더로** \> **더 표시를 선택합니다.**

3. 메시지를 닫습니다.

::: moniker-end

::: moniker range="o365-21vianet"

1. 메시지를 선택하여 열습니다.

2. **언더로** \> **더 표시를 선택합니다.**

3. 메시지를 닫습니다.

::: moniker-end

## <a name="archive-and-restore"></a>보관 및 복원

사용자와는 다른 메시지가 표시되거나 이미 해당 메시지에 대한 행동을 한 경우 메시지를 보관하여 메시지 센터 보기에서 제거할 수 있습니다. 메시지 센터에 표시하는 보기는 사용자 계정과 관련이 있으므로 보기에서 보관해도 다른 관리자에게는 영향을 주지 않습니다.

::: moniker range="o365-worldwide"

- 메시지 센터의 기본 페이지에서 메시지를 선택한 다음 메시지  목록 위에 있는 보관을 선택합니다.

- 메시지를 열고 메시지 아래쪽에 **있는** 보관을 선택합니다.

보관된 메시지를 다시 얻아야 하나요? 문제 없어요.
  
1. 메시지 **센터의** 맨 위에 있는 보관된 메시지 탭을 선택합니다. 보관된 메시지 목록이 나타납니다. 

2. 메시지를 선택하고 **복원을** 선택하면 메시지가 활성 메시지로 복원됩니다.

::: moniker-end

::: moniker range="o365-germany"

메시지를 보관하는 방법에는 두 가지가 있습니다.
  
- 메시지 ID 오른쪽에 나타나는 X를 선택합니다. 메시지가 목록에서 제거되었습니다. Or,

- 메시지를 열고 메시지 맨 위에 **있는** 보관을 선택합니다.

해지된 메시지를 다시 수신해야 하나요? 문제 없어요.
  
1. 보기 드롭다운 **메뉴에서** **보관된** 메시지 필터를 선택합니다. 보관된 메시지 목록이 나타납니다.

2. 메시지 **ID** 오른쪽으로 복원을 선택하면 메시지가 활성 메시지로 복원됩니다.

::: moniker-end

::: moniker range="o365-21vianet"

메시지를 보관하는 방법에는 두 가지가 있습니다.
  
- 메시지 ID 오른쪽에 나타나는 X를 선택합니다. 메시지가 목록에서 제거되었습니다. Or,

- 메시지를 열고 메시지 맨 위에 **있는** 보관을 선택합니다.

보관된 메시지를 다시 얻아야 하나요? 문제 없어요.
  
1. 보기 드롭다운 **메뉴에서** **보관된** 메시지 필터를 선택합니다. 보관된 메시지 목록이 나타납니다.

2. 메시지 **ID** 오른쪽으로 복원을 선택하면 메시지가 활성 메시지로 복원됩니다.

::: moniker-end

## <a name="track-your-message-center-tasks-in-planner"></a>Planner에서 메시지 센터 작업 추적

Microsoft 365 서비스 변경 내용에 대해 조치 가능한 많은 정보가 Microsoft 365 메시지 센터에 도착합니다. 작업을 완료해야 하는 변경 내용, 완료할 각 작업의 경우, 언제, 누가 추적해야 하는지 추적하기가 어려울 수 있습니다. 해당 사항을 메모하고 추후 확인하기 위해 태그를 지정할 수 있습니다. Microsoft 365 관리 센터에서 Microsoft Planner로 메시지를 동기화할 때 이 모든 작업을 할 수 있습니다. 자세한 내용은 [Planner에서 메시지 센터 작업 추적을 참조하세요.](https://docs.microsoft.com/office365/planner/track-message-center-tasks-planner)


메시지 센터에 대한 개요는 [Microsoft 365의 메시지 센터를 참조하세요.](message-center.md) 또는 메시지 센터 게시물에 대해 기계 번역을 사용하도록 언어 기본 설정을 설정하는 방법에 대한 자세한 내용은 메시지 센터 게시물에 대한 언어 [번역을 참조하세요.](language-translation-for-message-center-posts.md) 실시간 서비스 상태 정보 및 메시지 센터 통신을 받을 수 있는 다른 방법을 프로그래밍하고자 하는 경우 [Microsoft 365 서비스 통신 API 개요를 참조하세요.](https://go.microsoft.com/fwlink/p/?linkid=848507)
