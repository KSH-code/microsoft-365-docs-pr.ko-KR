# <a name="message-center"></a>메시지 센터


새 기능 및 변경된 기능, 계획된 유지 관리 또는 기타 중요한 공지 사항을 포함하여 예정된 변경 내용을 추적하기 위해 메시지 센터로 <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">이동하십시오.</a> 
  
메시지 센터를 열기 위해

::: moniker range="o365-worldwide"

- 관리 센터에서 상태  메시지 > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">센터로 이동하세요.</a>

::: moniker-end

::: moniker range="o365-germany"

- 관리 <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">센터에서</a>상태 메시지  > **센터로 이동하십시오.**

::: moniker-end

::: moniker range="o365-21vianet"
 
- 관리 <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">센터에서</a>상태 메시지  > **센터로 이동하십시오.**

::: moniker-end

모바일 장치에서 [Microsoft 365](https://go.microsoft.com/fwlink/p/?linkid=627216) 관리 앱을 사용하여 메시지 센터를 볼 수도 있습니다. 이는 푸시 알림으로 최신을 유지 하는 좋은 방법입니다.

### <a name="messages"></a>메시지

메시지 센터에서는 모든 활성 메시지의 보기를 표 형식으로 제공합니다. 기본적으로 목록의 맨 위에 가장 최근 메시지가 표시됩니다. 탭을 선택하여 모든 활성 **메시지,** 중요도 높은 **메시지,** **Unread 메시지** 및 닫은 메시지를 볼 수 **있습니다.** 또한 필터 드롭다운 메뉴를 사용하여 특정 메시지 범주의 모든 메시지를 볼 수 있습니다. 

다음은 각 열에 표시하는 정보에 대한 간략한 개요입니다. 정렬 순서를 변경하려면 열 제목을 선택합니다.
  
|**열**|**설명**|
|:-----|:-----|
|확인 표시  <br/> |열 제목 행에서 확인 표시를 선택하면 현재 표시된 모든 메시지가 선택됩니다. 하나 이상의 메시지 옆에 있는 확인 표시를 선택하면 해당 메시지에 대해 작업을 수행하게 됩니다.  <br/> |
|응용 프로그램 아이콘 <br/> |아이콘은 메시지가 적용되는 응용 프로그램을 나타냅니다.<br/> |
|메시지 제목  <br/> |메시지 제목은 예정된 변경 내용에 대한 간략한 설명입니다. 전체 제목이 표시되지 않는 경우 커서를 위에 대고 전체 제목이 팝업 상자에 나타납니다.  <br/> |
|기타 옵션 <br/> |추가 옵션을 사용하면 메시지를 삭제하거나 읽거나 읽지 않은 것으로 표시하거나 다른 관리자와 공유할 수 있습니다. 닫은 메시지를 복원하려면 닫은 메시지  탭을 선택하고 메시지 옆의 확인 표시를 선택한 다음 **복원을 선택합니다.** <br/> |
|주 업데이트 <br/> |이 열의 느낌표는 주요 업데이트를 나타냅니다. <br/> |
|행위자  <br/> |특정 기한까지 작업을 수행해야 하는 변경을 수행하고 있는 경우 여기에서만 날짜가 있습니다. 열로는 이 **법을** 거의 사용하지 않습니다. 여기에 있는 것이 있는 경우 특히 주의해야 합니다.  <br/> |
|범주  <br/> | 메시지는 다음 세 가지 범주 중 하나에 의해 식별됩니다. <br/><br/> **문제 방지** 또는 해결: 조직에 영향을 주는 알려진 문제를 알리며 서비스 중단을 방지하기 위해 조치를 취해야 할 수 있습니다. 문제 방지 또는 해결은 문제를 방지하라는 메시지를 표시하기 때문에 서비스 상태 메시지와 다릅니다. <br/> <br/> **변경 계획**: 서비스 중단을 방지하기 위해 행동해야 할 수 있는 Microsoft 365의 변경 내용을 알릴 수 있습니다. 예를 들어 시스템 요구 사항의 변경 내용이나 제거되는 기능에 대해 알려 드립니다. 관리자가 서비스를 정상적으로 실행하기 위해 작업해야 하는 변경에 대해 최소 30일 전에 알립니다. <br/> <br/> **최신 정보:** 조직에서 켜고 있는 새로운 기능 또는 업데이트된 기능에 대해 알릴 수 있습니다. 기능은 일반적으로 [Microsoft 365](https://go.microsoft.com/fwlink/?linkid=2070821)로드맵에서 먼저 발표됩니다. <br/><br/>또한 서비스 수준 계약에 따라 계획된 유지 관리에 대해 알 수 있습니다. 계획된 유지 관리로 인해 사용자 또는 사용자가 Microsoft 365, 특정 기능 또는 전자 메일 또는 비즈니스용 OneDrive와 같은 서비스에 액세스할 수 없는 다운 시간이 될 수 있습니다.  <br/> |
|마지막 업데이트  <br/> |메시지가 게시되거나 마지막으로 업데이트된 날짜입니다.  <br/> |
|메시지 ID  <br/> |Microsoft는 메시지 ID로 메시지 센터 게시물을 추적합니다. 피드백을 제공하려는 경우 또는 특정 메시지에 대한 지원에 문의하는 경우 이 ID를 참조할 수 있습니다.  <br/> |

메시지 [센터에서](manage-messages.md) 메시지를 관리하여 메시지로 할 수 있는 작업과 관련한 자세한 정보를 얻습니다.
  
### <a name="major-updates"></a>주요 업데이트

메시지 센터의 맨 위에 있는  중요도 높은 탭을 선택하여 주요 업데이트를 검토할 수 있습니다.

주요 업데이트는 작업이 필요하고 다음을 포함할 수 있는 경우 최소 30일 전에 전달됩니다.
  
- 받은 편지함, 모임, 위임, 공유 및 액세스와 같은 일상적인 생산성에 대한 변경 사항

- 사용자 지정 기능에 영향을 줄 수 있는 테마, 웹 파트 및 기타 구성 요소의 변경 내용

- 저장소, 규칙 수, 항목 또는 기간과 같이 표시되는 용량으로 증가 또는 감소

- 제품 브랜랜드에 대한 변경 사항:

  - 최종 사용자 혼동을 일으킬 수 있습니다.

  - 지원 센터 프로세스 및 참조 자료가 변경되거나

  - URL 변경

- 새 서비스 또는 응용 프로그램

- 관리자 작업을 요구하는 변경 내용(문제 방지 또는 해결 배타적)

- 데이터가 저장되는 위치에 대한 변경 내용
  
### <a name="preferences"></a>기본 설정

관리가 조직 전체에 배포된 경우 모든 Microsoft 365 서비스에 대한 게시물을 원하거나 볼 필요가 없습니다. 각 관리자는 다음을 할 수 있습니다.

- 메시지 센터에 표시되는 메시지를 제어하는 기본 설정을 설정할 수 있습니다.
- 메시지 필터링
- 모든 메시지의 주간 다이제스트, 주요 업데이트에 대한 전자 메일 및 데이터 개인 정보 메시지에 대한 전자 메일을 수신할 수 있도록 전자 메일 기본 설정을 설정하세요.  

::: moniker range="o365-worldwide"

1. 메시지 **센터 맨** 위에 있는 기본 설정 편집을 선택합니다.

2. 모니터링할 각 서비스에 대해 확인란이 선택되어 있는지 확인란을 선택해야 합니다. 메시지 센터 보기에서 필터링할 서비스의 확인란 선택을 취소합니다.

3. 다이제스트 전자 메일은 기본적으로 설정되어 있으며 기본 전자 메일 주소로 전송됩니다. 주간 다이제스트 수신을 중지하기 위해 내 메시지의 주간 전자 메일 **다이제스트 보내기** 확인란의 선택을 취소합니다. <br/><br/>주요 업데이트에 대한 전자 메일 알림은 별도의 컨트롤입니다. 주 업데이트에 대한 전자 메일 알림을 받으시고 싶은 경우 주 업데이트에 대한 전자 메일 보내기가 **선택되어** 있는지 확인합니다. 확인란의 선택을 취소하여 주요 업데이트에 대한 전자 메일 수신을 중지합니다. <br><br/>데이터 개인 정보  메시지에 대한 별도의 전자 메일 알림을 받으시고 데이터 개인 정보 메시지의 전자 메일 보내기 옆에 있는 확인란을 선택합니다(데이터 개인 정보 메시지는 주간 다이제스트에 포함되지 않음). <br><br/>기본 전자 메일 주소를 선택하거나 선택 취소할 수 있지만 변경할 수 없습니다. 데이터 개인 정보 메시지 전자 메일을 보낼 다른 전자  메일 주소를 지정하려면 다른 전자 메일 주소 확인란을 선택하고 전자 메일을 보낼 다른 전자 메일 주소를 입력합니다. 두 명 이상의 사용자가 데이터 개인 정보 전자 메일을 받을 수 있는 경우 Microsoft 365 그룹 또는 메일 그룹의 전자 메일 주소를 입력합니다.
