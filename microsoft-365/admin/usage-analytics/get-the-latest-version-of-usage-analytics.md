# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>최신 버전의 Microsoft 365 사용 현황 분석 다운로드

템플릿 앱은 1년에 여러 번 새 데이터 또는 새로운 시각화로 새로 고칠 수 있습니다. 기존 인스턴스는 계속 작동하지만 최신 버전을 사용하려면 새 인스턴스를 만들어야 합니다. 사용자 지정을 새 인스턴스에 적용해야 합니다. Microsoft [365 사용 현황 분석 사용 설정을 참조하세요.](enable-usage-analytics.md)
