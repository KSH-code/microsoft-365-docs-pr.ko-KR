# <a name="privacy-and-security-in-basic-mobility-and-security"></a>기본 이동성 및 보안의 개인 정보 및 보안

기본 모바일 및 보안은 조직에서 모바일 장치를 관리하고 보호하는 데 도움이 되는 Microsoft Intune에서 제공하는 클라우드 기반 서비스입니다. 기본 모바일 및 보안을 활성화한 후 모바일 장치 관리 정책을 만들 수 있습니다. 그런 다음 조직의 라이선스가 부여된 Microsoft 365 사용자가 등록한 모바일 장치에 이러한 정책을 배포할 수 있습니다.

Microsoft Intune은 각 관리되는 장치의 준수 상태에 대한 정보를 Microsoft 365로 전송한 다음 조직의 관리되는 장치가 설정된 정책을 기반으로 준수하는지 여부를 표시하는 보고서를 생성할 수 있습니다. 개인 정보 및 보안에 대한 Microsoft의 약속에 대한 자세한 내용은 [Microsoft 보안 센터를 참조합니다.](https://www.microsoft.com/trust-center)
