# <a name="overview-remove-a-former-employee-and-secure-data"></a>개요: 이전 직원 및 보안 데이터 제거

"직원이 조직을 떠날 때 데이터를 보호하고 액세스를 보호하기 위해 어떻게 해야 합니까?" 이 문서 시리즈는 Microsoft 365 대한 액세스를 차단하는 방법, 데이터를 보호하기 위해 취해야 할 단계 및 다른 직원이 데이터에 액세스할 수 있도록 하는 방법에 대해 설명합니다.

:::image type="content" source="../../media/delete-user-overview.png" alt-text="스크린샷: 이전 직원을 제거하기 위한 개요 단계":::

## <a name="before-you-begin"></a>시작하기 전에

이 솔루션의 단계를 완료하려면 전역 관리자여야 합니다.

## <a name="solution-remove-a-former-employee"></a>해결: 이전 직원 제거

> [!IMPORTANT]
> 이 솔루션의 단계에 번호가 매겨져 있고 정확한 순서를 사용하여 솔루션을 완료할 필요는 없지만 이 방식으로 단계를 수행하는 것이 좋습니다.

:::image type="content" source="../../media/delete-user-account.png" alt-text="스크린샷: 조직에서 이전 직원을 제거하기 위한 단계":::

|[2단계 - 이전 직원의 사서함 내용 저장](remove-former-employee-step-2.md) <br/> |이는 직원의 업무를 인수할 사람이거나 소송이 있는 경우 유용합니다. <br/> |

|[4단계 - 다른 직원에게 OneDrive 및 Outlook 데이터에 대한 액세스 권한을 부여합니다.](remove-former-employee-step-6.md) <br/> |사용자의 라이선스만 제거하고 계정을 삭제하지 않으면 30일이 지난 후에도 사용자의 OneDrive에 있는 콘텐츠에 계속 액세스할 수 있습니다. <br/><br/> 계정을 삭제하기 전에 OneDrive 액세스하고 다른 사용자에게 Outlook. 직원의 계정을 삭제하면 OneDrive 및 Outlook 콘텐츠는 **30일** 동안 유지됩니다. 그러나 30일 동안 사용자의 계정을 복원하고 해당 콘텐츠에 액세스할 수 있습니다. 사용자의 계정을 복원하면 OneDrive 및 Outlook 콘텐츠는 30일 후에도 계속 액세스할 수 있습니다. <br/> |
|[5단계 - 이전 직원의 모바일 장치를 지우고 차단합니다.](remove-former-employee-step-4.md) <br/> |휴대폰 또는 태블릿에서 비즈니스 데이터를 제거합니다.  <br/> |
|[6단계 - 이전 직원으로부터 Microsoft 365 라이선스를 제거하고 삭제합니다.](remove-former-employee-step-7.md) <br/> |라이선스를 제거하면 다른 사람에게 해당 라이선스를 할당할 수 있습니다. 또는 다른 사람을 고용할 때까지 라이선스 비용을 지불하지 않도록 라이선스를 삭제할 수 있습니다.  <br/><br/> 라이선스를 제거하거나 삭제하면 사용자의 이전 전자 메일, 연락처 및 일정이 **30일** 간 보존된 후 영구적으로 삭제됩니다. 라이선스를 제거하거나 삭제하되 계정은 삭제하지 않으면 30일이 지난 후에도 사용자의 OneDrive에 있는 콘텐츠에 계속 액세스할 수 있습니다.  <br/> |
