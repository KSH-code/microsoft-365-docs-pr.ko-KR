# <a name="about-the-exchange-online-admin-role"></a>Exchange Online 관리자 역할

Microsoft 365를 관리할 수 [](assign-admin-roles.md) 있도록 Exchange 관리 센터에서 조직의 전자 메일 및 사서함을 관리할 수 있는 사용 권한을 [사용자에게 할당할 수 있습니다.](https://go.microsoft.com/fwlink/p/?LinkID=271807) 이 작업을 위해 Exchange 관리자 역할에 할당합니다.
  
 **팁:** Exchange 관리자 역할에 다른 사람을 할당할 때 서비스 관리자 역할에도 할당합니다. 이렇게 하면 Microsoft 365 관리 센터에서 Exchange Online 서비스의 상태와 같은 중요한 정보를 보고 알림을 변경 및 해제할 수 있습니다.
  
## <a name="before-you-begin"></a>시작하기 전에

다음은 사용자가 Exchange 관리자 역할에 할당될 때 수행할 수 있는 몇 가지 주요 작업입니다.
  
- [사용자 사서함에서 지운 편지함 복구 - 관리자 도움말](https://docs.microsoft.com/microsoft-365/enterprise/recover-deleted-items-in-a-mailbox)

- 조직의 사서함에 대한 보관 및 삭제 [정책을 설정합니다.](https://docs.microsoft.com/microsoft-365/compliance/set-up-an-archive-and-deletion-policy-for-mailboxes)

- 사서함 공유 정책과 같은 사서함 기능 설정: 사용자가 조직 외부의 다른 사용자와 일정 및 연락처 정보를 공유하는 방법

- "다른 사람[사서함에](give-mailbox-permissions-to-another-user.md#send-email-from-another-users-mailbox)대해 "다른 사람 대신 보내기" 및["대신](give-mailbox-permissions-to-another-user.md#send-email-on-behalf-of-another-user)보내기" 대리인을 설정합니다. 예를 들어 임원은 비서가 대신 메일을 보낼 수 있도록 하려는 경우를 예로 들 수 있습니다.

- [사용자 그룹이](../email/create-a-shared-mailbox.md) 공통 전자 메일 주소에서 전자 메일을 모니터링하고 보낼 수 있도록 공유 사서함을 생성합니다.

- [조직의 스팸 방지 보호 및](https://docs.microsoft.com/microsoft-365/security/office-365-security/anti-spam-protection) 맬웨어 필터를 전자 메일로 보내기

- Microsoft 365 그룹 관리

## <a name="exchange-online-role-groups"></a>Exchange Online 역할 그룹

대규모 조직이 있는 경우 Exchange 관리자가 사용자를 Exchange 역할 그룹에 할당할 수 있습니다. 관리자가 역할 그룹에 사용자를 추가하면 사용자는 해당 그룹의 구성원만 수행할 수 있는 특정 비즈니스 기능을 수행할 수 있는 사용 권한을 얻습니다.
  
 예를 들어 Exchange 관리자는 사서함에서 특정 기준을 충족하는 데이터를 검색할 수 있도록 검색 관리 역할 그룹에 누군가를 할당할 수 있습니다. 자세한 내용은 [Exchange Online의 사용 권한](https://docs.microsoft.com/exchange/permissions-exo/permissions-exo) 및 역할 그룹 [관리를 참조하세요.](https://docs.microsoft.com/exchange/manage-role-groups-exchange-2013-help)
