# <a name="let-users-reset-their-own-passwords"></a>사용자가 암호를 직접 재설정할 수 있도록 허용

Microsoft 365 관리자는 사람들이 [셀프 서비스 암호 재설정 도구를](https://go.microsoft.com/fwlink/p/?LinkId=522677) 사용하도록 허용하여 암호를 재설정할 필요가 없습니다. 관리자의 업무를 덜 수 있습니다!
  
## <a name="before-you-begin"></a>시작하기 전에
  
- Microsoft 365 비즈니스, 교육 또는 비영리 유료 요금제와 함께 클라우드 사용자를 위한 셀프 서비스 암호 재설정을 **무료로** 받을 수 있습니다. 그것은 Microsoft 365 재판과 함께 작동하지 않습니다.

- Azure를 사용합니다. 이러한 단계를 수행할 때 자동으로 Azure의 이 기능을 **무료** 로 이용할 수 있습니다. 다른 Azure 기능을 사용하지 않으면 셀프 서비스 암호 재설정을 사용하는 비용이 부과되지 않습니다.

- **온-프레미스 액티브 디렉터리를 사용하는 경우** 위의 두 점이 적용되지 않습니다. 대신 이 설정을 설정할 수 있지만 **Azure AD에 대한 유료 구독이 필요합니다Premium.**

회사, 학교 또는 비영리용 암호 만료 정책을 설정하는 사용자를 위한 문서입니다. 이 단계를 완료하려면 Microsoft 365 관리자 계정으로 로그인해야 합니다. [관리자 계정은 무엇입니까?](../../business-video/admin-center-overview.md)

이러한 단계를 수행하려면 [전역 관리자 또는 암호 관리자여야](about-admin-roles.md) 합니다.

## <a name="steps-let-people-reset-their-own-passwords"></a>단계: 사람들이 자신의 암호를 재설정하도록 합니다.

다음 단계에서는 비즈니스의 모든 사용자에 대해 셀프 서비스 암호 재설정을 켭니다.

1. 관리 <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">센터에서</a>설정 조직   >  **설정** 페이지로 이동합니다.

2. **Org 설정** 페이지 상단에서 보안 **& 개인 정보 보호** 탭을 선택합니다.
  
3. **셀프 서비스 암호 재설정을 선택합니다.**

4. **셀프 서비스 암호 재설정에서** **Azure 포털로 이동을 선택하여 셀프 서비스 암호 재설정을 켭니다.**

5. 왼쪽 탐색 창에서 **사용자를** 선택한 다음 사용자가 **| 모든 사용자** 페이지, **암호 재설정을 선택합니다.**
  
6. **속성** 페이지에서 **모든** 것을 선택하여 비즈니스의 모든 사용자가 사용하도록 설정한 다음 **저장을** 선택합니다.
  
7. 사용자가 로그인하면 나중에 암호를 재설정하는 데 도움이 되는 추가 연락처 정보를 입력하라는 메시지가 표시됩니다.
