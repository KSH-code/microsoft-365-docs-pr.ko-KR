# <a name="create-and-configure-a-prebuilt-model-in-microsoft-sharepoint-syntex"></a>Microsoft 365에서 미리 작성된 모델 SharePoint Syntex

> [!NOTE]
> 이 기능의 콘텐츠는 현재 개발 중입니다.

문서 및 문서의 구성 요소를 인식하기 위해 미리 작성된 모델입니다. 또한 이미지 파일 및 PDF 파일에서도 사용할 수 있습니다. 미리 빌드된 모델을 라이브러리에 있는 즉시 적용할 수 있습니다. 추가 교육이나 레이블 지정이 필요하지 않은 경우 교육 파일이 필요하지 않습니다.

미리 작성된 모델을 사용자 지정하고 검토하여 문서에서 작동하는 방법을 확인한 다음 문서 또는 명명 규칙에 맞게 이름을 사용자 지정할 수도 있습니다. 적용할 수 없는 제안된 구성 요소를 해제할 수도 있습니다.
