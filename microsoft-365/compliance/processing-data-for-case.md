# <a name="work-with-processing-errors-in-advanced-ediscovery"></a>Advanced eDiscovery에서 오류 처리 작업

*처리는* 이미지 파일의 파일 식별, 포함된 문서 및 첨부 파일 확장, 텍스트 추출 및 OCR(광학 문자 인식)과 해당 콘텐츠의 후속 인덱싱 프로세스입니다.  

원본 탭의 사례에 보호자 및 비관리자 데이터 원본을 추가하면  Microsoft 365의 부분적으로 인덱싱된 모든 항목이 완전히 검색 가능해지기 위해 처리됩니다. 마찬가지로, Microsoft 365 및 비 Microsoft 365 데이터 원본의 검토 집합에 콘텐츠를 추가하면 이 콘텐츠도 처리됩니다.

Advanced  eDiscovery의 처리 탭은 다양한 처리 시나리오에 대한 고급 인덱싱 상태에 대한 정보를 제공합니다.

자세한 내용은 다음 문서를 참조하세요.

- [보유자 데이터의 고급 인덱싱](indexing-custodian-data.md)

- [데이터를 처리할 때 오류 수정](error-remediation-when-processing-data-in-advanced-ediscovery.md)

- [단일 항목 오류 수정](single-item-error-remediation.md)
