# <a name="ediscovery-solutions-in-microsoft-365"></a>Microsoft 365의 eDiscovery 솔루션

전자 검색 또는 eDiscovery는 법률 소송에서 증거로 사용할 수 있는 전자 정보를 식별하고 전달하는 프로세스입니다. Microsoft 365의 eDiscovery 도구를 사용하여 Exchange Online 사서함, Microsoft 365 그룹, Microsoft Teams, SharePoint Online 및 비즈니스용 OneDrive 사이트, 비즈니스용 Skype 대화 및 비즈니스용 Skype Yammer 검색할 수 있습니다. 콘텐츠 검색 도구를 사용하여 동일한 eDiscovery 검색에서 사서함 및 사이트를 검색할 수 있습니다. 또한 Core eDiscovery 사례를 사용하여 사서함 및 사이트에서 찾은 콘텐츠를 식별, 보류 및 내보낼 수 있습니다. 조직에 Office 365 E5 또는 Microsoft 365 E5 구독(또는 관련 E5 추가 기능 구독)이 있는 경우 Microsoft 365의 Advanced eDiscovery 솔루션을 사용하여 상주를 추가로 관리하고 콘텐츠를 분석할 수 있습니다.
  
Microsoft 365는 다음과 같은 eDiscovery 도구를 제공합니다.
  
- [콘텐츠 검색](#content-search)

- [핵심 eDiscovery](#core-ediscovery)

- [고급 eDiscovery](#advanced-ediscovery)

## <a name="advanced-ediscovery"></a>Advanced eDiscovery

Microsoft 365의 Advanced *eDiscovery 솔루션(Advanced eDiscovery v2.0)은* Microsoft 365의 기존 eDiscovery 및 분석 기능을 빌드합니다. 이 eDiscovery 솔루션은 조직의 내부 및 외부 조사에 응답하는 콘텐츠를 보존, 수집, 검토, 분석 및 내보내기 위한 종단 간 워크플로를 제공합니다. 또한 법률 팀은 보유자 및 전체 법적 보유 알림 워크플로를 관리하여 사례에 관련된 보유자와 통신할 수 있습니다.

|:-----|:-----|
