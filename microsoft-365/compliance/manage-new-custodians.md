<span data-ttu-id="2a02e-167">양도인은 사례가 마감되거나, 양도인이 더 이상 사례에 대한 콘텐츠를 보존할 의무가 없는 경우 또는 해당 사례와 더 이상 관련이 없는 것으로 판단되는 경우에 릴리스됩니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-167">A custodian is released in situations where a case is closed, the custodian is no longer under obligation to preserve content for a case, or when the custodian is deemed to no longer be relevant to the case.</span></span> 

<span data-ttu-id="2a02e-168">보류 알림이 게시된 후 보유자 해제 시, 보유자에 대한 릴리스 알림이 전송됩니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-168">If you release a custodian after a hold notice was published, a release notice will be sent to the custodian.</span></span> <span data-ttu-id="2a02e-169">또한 보유자와 연결된 데이터 원본에 배치된 보류도 모두 제거됩니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-169">Additionally, any holds placed on data sources that were associated with the custodian are removed.</span></span> <span data-ttu-id="2a02e-170">법적 보유 알림이 발급되지 않은 자동 보류를 설정한 경우, 릴리스 알림은 전송되지 않지만 해당 보유자에 연결된 데이터 원본에 대한 보류는 제거됩니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-170">If the custodian was placed on a *silent hold*, where they weren't issued any legal hold notifications, a release notice will not be sent but any holds placed on data sources that were associated with that custodian are removed.</span></span>

<span data-ttu-id="2a02e-171">양도인을 해제합니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-171">To release a custodian:</span></span> 

1. <span data-ttu-id="2a02e-172">Advanced  **eDiscovery를 > eDiscovery를 통해 eDiscovery로** 이동한 후 사례를 여는 것이 가장 까다로우며,</span><span class="sxs-lookup"><span data-stu-id="2a02e-172">Go to  **eDiscovery > Advanced eDiscovery** and open the case.</span></span>

2. <span data-ttu-id="2a02e-173">원본 **탭을** 클릭합니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-173">Click the **Sources** tab.</span></span>

3. <span data-ttu-id="2a02e-174">**Custodians** 페이지에서 사례에서 릴리스되는 양도인을 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-174">On the **Custodians** page, and then select the custodian who is being released from the case.</span></span>

4. <span data-ttu-id="2a02e-175">플라이아웃 페이지에서 **Custodian 릴리스를 클릭합니다.**</span><span class="sxs-lookup"><span data-stu-id="2a02e-175">On the flyout page, click **Release custodian**.</span></span>

   <span data-ttu-id="2a02e-176">보유자에 연결된 데이터 원본에 보류가 적용되면 보류가 제거되고 다른 Advanced eDiscovery 사례와 연결된 다른 모든 보류가 계속 적용됨을 설명하는 경고 페이지가 표시됩니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-176">A warning page is displayed explaining that if a hold is placed on a data source associated with the custodian, the hold will be removed, and that any other hold associated with a different Advanced eDiscovery case will still apply.</span></span> <span data-ttu-id="2a02e-177">여기에는 다른 유형의 보존 및 보존 기능(예: Microsoft 365 보존 정책)이 포함됩니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-177">That includes other types of preservation and retention features (such as a Microsoft 365 retention policy).</span></span>

5. <span data-ttu-id="2a02e-178">**예를** 클릭하여 양도인을 해제할지 확인할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-178">Click **Yes** to confirm that you want to release the custodian.</span></span> 

    <span data-ttu-id="2a02e-179">보유자 탭에서 이  사용자의 상태가 릴리스로 설정되어  있으며 플라이아웃 페이지의 보류 상태가 **False로 변경됩니다.** </span><span class="sxs-lookup"><span data-stu-id="2a02e-179">The status for this user on the **Custodians** tab is set to **Released** and the **Hold status** on the flyout page is changed to **False**.</span></span> 

> [!NOTE]
> <span data-ttu-id="2a02e-180">여러 법적 사건에 양도권이 동시에 관련될 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-180">A custodian might be simultaneously involved in several legal cases.</span></span> <span data-ttu-id="2a02e-181">보유자도 사례에서 릴리스될 경우 다른 문제의 보류 및 알림은 영향을 하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="2a02e-181">When a custodian is released from a case, the holds and notifications across other matters won't be impacted.</span></span>
