# <a name="plan-for-security-amp-compliance"></a>보안 준수 &amp; 계획

보안 및 규정 준수 관리는 파트너 관계입니다. Microsoft는 데이터, ID 및 장치를 보호할 책임이 있으며 Microsoft는 Microsoft 365 보호합니다. EMS(Microsoft 365 및 Enterprise Mobility + Security)를 함께 사용하여 조직에 적합한 수준의 보호를 달성할 수 있습니다.
  
## <a name="step-1-review-capabilities"></a>1단계: 기능 검토

정보 보호 포스터의 정보 보호 Office 365 방향을 안내합니다. 
  
[데이터 개인 정보 보호 규정에 대한 정보 보호를 Microsoft 365](../solutions/information-protection-deploy.md)
  
## <a name="step-2-check-your-secure-score"></a>2단계: 보안 점수 확인

구독을 설정한 Microsoft 365 시작 점수를 기록해 넣습니다. 보안 점수는 점수를 높이기 위해 취할 수 있는 구성 제안을 제공합니다. 목표는 사용자의 생산성에 부정적인 영향을 주지 않는 환경을 보호하기 위해 취할 수 있는 기회를 인식하는 것입니다.
  
- [보안 점수 Office 365 소개](../security/defender/microsoft-secure-score.md)
    
## <a name="step-3-plan-access-protection-for-identity-and-devices"></a>3단계: ID 및 장치에 대한 액세스 보호 계획

데이터 및 서비스에 Microsoft 365 보호하는 것은 사이버 공격으로부터 보호하고 데이터 손실을 방지하는 데 중요합니다.
