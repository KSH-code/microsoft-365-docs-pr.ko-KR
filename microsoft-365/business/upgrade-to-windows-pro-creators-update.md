# <a name="upgrade-windows-devices-to-windows-10-pro"></a>Windows 장치를 Windows 10 Pro로 업그레이드

Windows 장치를 Windows 10 Pro로 업그레이드하는 방법을 짧은 비디오를 시청합니다.<br><br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3t58j] 

이 비디오가 도움이 된 경우에는 [소규모 비즈니스와 Microsoft 365를 처음 사용하는 사용자를 위한 전체 교육 시리즈](https://support.microsoft.com/office/6ab4bbcd-79cf-4000-a0bd-d42ce4d12816)를 참조하세요.

## <a name="upgrade-to-windows-10-pro"></a>Windows 10 Pro로 업그레이드
  
Windows 10 Pro로 업그레이드하려면 몇 가지 옵션이 있습니다. 다음을 수행할 수 있습니다.
    
- [Microsoft](https://go.microsoft.com/fwlink/?LinkID=836951 ) 소프트웨어 다운로드 사이트에서 업그레이드를 설치합니다. 로그인한 장치가 업데이트하려는 장치와 동일한 장치에 있는 경우 이 옵션을 &ndash; 선택합니다. 

    - 소프트웨어 다운로드 사이트에서 지금 업데이트를 **클릭하여** 디바이스를 Windows 10 Pro 크리에이터스 업데이트로 업그레이드합니다. 
    
- 미디어 만들기 도구를 [](https://go.microsoft.com/fwlink/?LinkID=836960) 사용하여 설치 미디어를 만들고 Windows 10 Pro 크리에이터스 업데이트 설치 미디어(USB 플래시 드라이브 또는 ISO 파일)를 만들어 사용중인 PC와 다른 &ndash; Windows 10을 설치합니다.

    - 도구를 사용하여 설치 미디어를 만드는 방법에 대한 지침을 읽어 읽습니다. 

> [!NOTE]
> Windows 7 Pro, Windows 8 Pro 또는 Windows 8.1 Pro를 실행하는 Windows 장치가 있는 경우 비즈니스용 Microsoft 365 구독에서 Windows Pro 10 업그레이드를 받을 수 있습니다.
    
비즈니스용 [Microsoft 365](set-up-windows-devices.md) 사용자를 위해 Windows 장치 설정을 참조하여 Windows 10 장치 설정을 완료합니다. 
  
비즈니스용 [Microsoft 365](set-up-mobile-devices.md) 사용자를 위한 모바일 장치 설정에서 Android 및 iOS 장치 설정을 완료합니다.
