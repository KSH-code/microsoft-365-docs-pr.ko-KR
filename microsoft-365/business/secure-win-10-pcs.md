# <a name="secure-windows-10-computers"></a>Windows 10 컴퓨터 보안

이 문서는 Microsoft 365 Business Premium에 적용됩니다.

[Microsoft](set-up.md) 365 Business Premium을 설정한 후 도난 및 바이러스 및 맬웨어와 같은 악의적인 위협으로부터 Windows 10 컴퓨터를 보호해야 합니다.

## <a name="to-secure-your-windows-10-computers"></a>Windows 10 컴퓨터를 보호하는 방법

1. 전역 관리자 자격 증명을 사용하여 [Microsoft 365 관리 센터](https://admin.microsoft.com)에 로그인합니다. 
2. 왼쪽 네비게이트에서 설치를 선택한 다음 로그인 및 보안에서 **Windows 10** 컴퓨터 보안 선택  **보기를** 선택하면 시작됩니다.
3. Windows **10** 컴퓨터 보안 페이지에서 모든 정보를 읽고 켜고 있는 설정 및 사용자 영향에 대해 이해합니다.

    페이지 맨 위에 있는 시작을 **선택합니다.**

4. Windows **10 컴퓨터** 보안 창에서 켜고 싶은 옵션을 선택합니다. 설정에 대한 자세한 내용은 [보안 Windows 10 장치를 참조하세요.](secure-windows-10-devices.md) 
    
    대부분의 조직에서 이 옵션을 사용하면 높은 수준의 보안을 제공할 수 있습니다. 그러나 조직에 더 복잡한 보안 요구가 있는 경우 미리 정의된 보안 기준을 사용하여 Windows 10 장치를 보호할 수도 있습니다. 자세한 내용은 Windows 10 장치에 대한 보안 [기준을 참조하세요.](https://docs.microsoft.com/mem/intune/protect/security-baselines)   

1. 설정 **적용을 선택합니다.**

    이러한 설정은 조직의 모든 사용자에게 적용됩니다. 서로 다른 보안 그룹에 대해 다른 정책을 설정하려면 [Windows 10 PC에](protection-settings-for-windows-10-pcs.md)대한 장치 보호 설정 설정을 참조하세요.
