# <a name="enable-user-support-features"></a>사용자 지원 기능 사용

자체 사용자 지원을 제공하거나 파트너와 협력하여 지원을 제공하려면 다음 단계를 수행하여 지원 공급자가 상승된 장치 액세스를 요청하거나 필요한 경우 Microsoft Managed Desktop 에스컬레이터할 수 있습니다.

1. 아직 계정이 없는 경우 사용자는 AAD(Azure Active Directory) 도메인과 동일한 Microsoft Managed Desktop 필요합니다.
2. 1단계의 사용자 계정을 AAD의 최신 Workplace Roles-Support **파트너** 보안 그룹에 추가합니다.
