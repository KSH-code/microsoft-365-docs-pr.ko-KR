# <a name="set-up-microsoft-managed-desktop-devices"></a>Microsoft Managed Desktop 장치 설정

새 디바이스와 기존 장치를 모두 사용할 수 Microsoft Managed Desktop.

## <a name="to-obtain-new-devices"></a>새 디바이스를 얻은 경우

승인된 장치 파트너 중 하나와 함께 작업하는 것이 좋습니다. Microsoft 계정 연락처와 협력하여 장치 파트너 관계를 설정하는 데 더 많은 도움을 줄 수 있습니다. 그러나 본질적으로 프로세스는 다음과 같습니다.

1. Shop Windows 10 Pro Business Devices 사이트에서 비즈니스용 디바이스를 필터링하여 현재 Microsoft Managed Desktop [목록을 검토합니다.](https://www.microsoft.com/windowsforbusiness/view-all-devices)
2. 규격 이미지와 함께 사용할 장치의 예를 하나 또는 몇 가지 주문합니다. 순서를 지정하려면 특정 [순서 단계가 필요할 수 있습니다.](../service-description/device-images.md)
3. [예제](validate-device.md) 디바이스의 유효성을 검사합니다.
5. 유효성 검사에 성공한 후 승인된 장치 파트너와 함께 작업하여 디바이스를 주문합니다.
6. 도착한 후 다음 중 하나를 합니다.
    - [직접 새 장치 등록](register-devices-self.md)
    - 파트너와 협력하여 디바이스 등록
7. [사용자가 디바이스를](get-started-devices.md) 사용할 Microsoft Managed Desktop 준비

## <a name="to-reuse-existing-devices"></a>기존 디바이스를 다시 사용

1. 기존 장치가 장치 요구 사항을 [충족하는지 확인합니다.](../service-description/device-requirements.md) 다운로드 가능한 준비 평가 [](../get-ready/readiness-assessment-downloadable.md) 검사를 사용하여 특정 장치가 필요한 요구 사항을 충족하는지 확인할 수 있습니다. 
2. 기존 디바이스를 다시 사용할 경우 다시 이미지해야 할 수 있습니다. 이미지 옵션에 대한 자세한 내용은 장치 이미지를 [참조하세요.](../service-description/device-images.md)
3. 다시 사용할 디바이스의 예를 하나 또는 몇 개 선택한 다음 [유효성을 검사합니다.](validate-device.md)
4. 유효성 검사가 성공한 후 다음 중 하나를 합니다.
    - [직접 기존의 장치 등록](register-reused-devices-self.md)
    - 파트너와 협력하여 디바이스 등록
5. [사용자가 디바이스를](get-started-devices.md) 사용할 Microsoft Managed Desktop 준비
