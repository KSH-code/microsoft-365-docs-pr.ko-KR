# <a name="set-up-microsoft-managed-desktop-devices"></a>Microsoft Managed Desktop 장치 설정

Microsoft Managed Desktop에서 새 디바이스와 기존 디바이스를 모두 사용할 수 있습니다.

## <a name="to-obtain-new-devices"></a>새 디바이스를 얻은 경우

승인된 장치 파트너 중 하나를 사용하여 작업하는 것이 좋습니다. Microsoft 계정 담당자와 협력하여 장치 파트너 관계를 설정하는 데 도움이 될 수 있습니다. 그러나 본질적으로 프로세스는 다음과 같습니다.

1. 현재 승인된 장치 [목록을 검토합니다.](../service-description/device-list.md)
2. 승인된 장치 파트너와 함께 작업하여 디바이스를 주문합니다.
3. 도착한 후 다음 중 하나를 합니다.
    - [직접 새 장치 등록](register-devices-self.md)
    - 파트너와 협력하여 디바이스 등록
4. [사용자가](get-started-devices.md) Microsoft Managed Desktop 디바이스를 사용할 수 있도록 준비

## <a name="to-reuse-existing-devices"></a>기존 디바이스를 다시 사용

1. 현재 승인된 [](../service-description/device-list.md) 장치 목록을 검토하여 Microsoft Managed Desktop에서 사용할 수 있도록 승인된 장치를 식별합니다. 기존 디바이스를 다시 사용할 경우 다시 이미 이미지해야 합니다.
2. 다음 중 하나를 합니다.
    - [직접 기존의 장치 등록](register-reused-devices-self.md)
    - 파트너와 협력하여 디바이스 등록
3. [사용자가](get-started-devices.md) Microsoft Managed Desktop 디바이스를 사용할 수 있도록 준비
