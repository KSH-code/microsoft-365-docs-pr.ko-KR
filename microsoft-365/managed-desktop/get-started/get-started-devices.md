# <a name="get-your-users-ready-to-use-devices"></a>사용자들이 장치를 사용할 수 있도록 준비시키기

사용자가 Microsoft Managed Desktop 디바이스를 시작하면 빠르고 쉽게 시작할 수 있습니다. 디바이스는 현재 버전의 디바이스로 미리 Windows 구성 및 앱은 사용자가 설치를 완료하면 클라우드에서 설치됩니다. 
 
이를 보다 쉽게 만들기 위해 초기 설치를 안내하는 가이드를 제공하며, 필요한 경우 설치 및 나중에 사용하기 위한 도움말 리소스를 제공합니다. 이 가이드를 사용자 지정하여 조직에 특정한 세부 정보를 포함할 수 있습니다. 그런 다음 디바이스와 함께 사용자에게 직접 가이드를 배포합니다. 

## <a name="to-prepare-the-guide"></a>가이드를 준비하기 위해

1. 가이드 다운로드: [Microsoft Managed Desktop - 장치](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-custom-v2.pdf)시작 .
2. PDF 파일을 열 수 있는 앱을 사용하여 조직과 관련된 세부 정보를 입력합니다.
    - 설치를 계속하기 위해 사용자가 연결해야 하는 네트워크의 이름(가이드의 3단계)
    - 조직의 Azure 테넌트 계정 이름(가이드의 4단계)
    - 조직의 내부 IT 지원에 대한 연락처 정보(두 번째 페이지 맨 위)
3. 편집된 PDF를 저장한 다음 사용자에게 배포합니다. 

## <a name="ready-to-use-guide"></a>"즉시 사용 가이드"

또한 사용자 지정할 필요가 없는 조직을 위한 보다 일반적인 버전의 가이드도 제공합니다.

가이드를 다운로드하면 됩니다. [Microsoft Managed Desktop - 디바이스 시작(사용 준비 완료)입니다.](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-v2.pdf)

이제 앱 배포를 진행할 준비가 완료되었습니다.
