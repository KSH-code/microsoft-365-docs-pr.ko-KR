# <a name="order-microsoft-managed-desktop-devices"></a>디바이스 Microsoft Managed Desktop 순서

승인된 장치 파트너 중 하나와 함께 작업하는 것이 좋습니다. Microsoft 계정 연락처와 협력하여 장치 파트너 관계를 설정하는 데 더 많은 도움을 줄 수 있습니다.

Microsoft Managed Desktop 더 이상 현재 승인된 장치 목록에서 장치 모델을 [사용할 필요는 없습니다.](../service-description/device-list.md) 2021년 5월 3일부로 승인된 제조업체의 모든 장치는 게시된 하드웨어 및 소프트웨어 요구 사항을 충족해야 합니다. 해당 목록에서 장치를 계속 사용할 수 있지만 비즈니스용 디바이스 쇼핑 사이트에서 Microsoft Managed Desktop [권장되는](https://www.microsoft.com/windowsforbusiness/view-all-devices) Windows 10 Pro 있습니다. 해당 사이트에서 필터로 필터링 영역의  기능을  확장한 다음 을 선택하여 권장 장치를 **Microsoft Managed Desktop.** 서비스에 특정 장치 모델을 처음 등록할 계획이면 언제든지 예제를 테스트하여 원하는 사용자 환경을 제공할 수 있도록 해야 합니다. 자세한 내용은 새 장치 [유효성 검사를 참조하세요.](../get-started/validate-device.md)
