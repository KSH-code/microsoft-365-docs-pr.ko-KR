# <a name="microsoft-cloud-for-enterprise-architects-illustrations"></a>엔터프라이즈 설계자용 Microsoft 클라우드 일러스트레이션

이러한 클라우드 아키텍처 포스터는 microsoft 365, Azure Active Directory (Azure AD), Microsoft Intune, Microsoft Dynamics 365 및 하이브리드 온-프레미스 및 클라우드 솔루션을 포함 하 여 Microsoft 클라우드 서비스에 대 한 정보를 제공 합니다. 

IT 의사 결정권자와 설계자는 이러한 리소스를 사용 하 여 작업 부하에 이상적인 솔루션을 확인 하 고 네트워킹, id 및 보안과 같은 핵심 인프라 구성 요소에 대 한 결정을 내릴 수 있습니다.
